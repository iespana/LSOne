Statement Line Dialog

Retail > End of day > Unposted Statements > Highlight statement line > Select Edit button

What is the view/dialog used for?

  • Edit counted amount for the statement line.
Fields & buttons Description
Store The store that you have selected in the main view.
Statement number The Statement number.
Terminal ID The Terminal ID.
Staff ID The Staff ID.
Tender The tender type, for example, Cash.
Transaction amount The transaction amount in total.
Banked amount The amount that has been put in the bank.
Safe amount The amount that has been put in the safe.
Counted amount The amount that has been counted.
Difference The difference between the amount counted and the total transaction amount.
