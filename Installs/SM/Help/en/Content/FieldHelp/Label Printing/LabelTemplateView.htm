Item/Customer Label Template View

Store setup > Forms and labels > Item/Customer label templates

What is the view used for?

  • To edit the layout and information printed on labels such as price shelf labels for items and address labels for customers.
Fields & buttons Description
Label ID Displays the ID of the Label
Context Item or Customer depending on which type of label is open in the view.
Name Edit the name of the label.
Template

Setup for the label.

Fields & buttons Description
Description Enter a descriptive text for the label for example .
Codepage Enter the code page character set for the print out. The label printer must support the selected code page.
Template

The template field is plain text that is sent to the printer (in raw mode). The template must therefore be in a form that the printer recognizes and can translate to print commands.

An example of this is the ZPL (Zebra Programming Language) or EPL (Eltron Programming Language) â€“ both for Zebra printers. An EPL programming manual can be found here: EPL2 Manual

The lines of the template code are sent directly to the printer so changing paper format on the printer does not work, labels will always be printed as portrait. One can how ever change the rotation of the lines printed in the template code. Instructions for how to do that can be found under Command reference - A (Ascii text) and B (Barcodes) in the manual provided above.

Any variables found in the template are substituted with actual values for the item being printed.
Select this button to get a list of variables that can be substituted for actual vales on the label.
Sample Enter a scanned image or a photo preview of a printed label by clicking on the edit button.
Edit Select this button to browse for an image to display as a preview of the label.
