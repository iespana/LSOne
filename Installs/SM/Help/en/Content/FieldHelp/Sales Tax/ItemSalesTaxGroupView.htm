Item Sales Tax Group View

General setup > Sales tax > Item sales tax group

What is the view/dialog used for?

  • View, add and edit item sales tax groups.

List of item sales tax groups

A list of all existing item sales tax groups

Fields & buttons Description
ID Displays the item sales tax group ID.
Description Displays the item sales tax group description.
Edit Highlight a line and select this button to edit the item sales tax group and open Edit item sales tax group dialog.
Add Select this button to create a new item sales tax group and open the New item sales tax group dialog.
Delete Highlight a line and select this button to delete an item sales tax group.
Codes in group

A list of all sales tax codes that are included in the selected item sales tax group in the upper pane.

Fields & buttons Description
Sales tax code Displays the sales tax code ID.
Description Displays the sales tax code description.
Percentage Displays the percentage value of the sales tax code.
Button:
View tax code		 Highlight a sales tax code line and select this button to view it by opening the Sales tax codes view.
Add Select this button to add or create a new sales tax code to the item sales tax group. This opens the New tax code for item sales tax group dialog.
Delete Highlight a sales tax code line and select this button to delete it.
