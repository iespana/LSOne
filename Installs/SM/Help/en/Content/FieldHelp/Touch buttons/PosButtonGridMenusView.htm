POS Button Grid Menus View

Store setup > Look and feel > POS button grid menus.

Upper Pane
Tab/Field name Description
ID Displays the POS button grid menu ID.
Description Displays the POS button grid menu description.
Columns The number of columns in the grid.
Rows The number of rows in the grid.
Style The style profile set to the grid.
Button:
Preview		 Click on a line and select this button to preview the POS button grid menu. Opens the Preview dialog.
Edit Click on a line and select this button to open POS menu header dialog.
Add Select this button to add a new POS button grid menu. Opens POS menu header dialog.
Delete Click on a line and select this button to delete.
Lower Pane: Menu Buttons
Tab/Field buttons Description
Key No The key number is the order the button appears in, from left to right, top to bottom. This can be changed by clicking on a line and selecting an arrow to move it up or down on the list. You can then preview the changes to the button grid menu by selecting the Preview button.
Description Displays the description of the button.
Operation Displays the operation performed on the POS when this button is used.
Parameter Displays the parameter of the operation (if operation has more than one possibility).
Style Displays the style profile that the button is set to if different to the button grid menu.
Arrow up The key number is the order the button appears in, from left to right, top to bottom. This can be changed by clicking on a line and selecting an arrow to move it up or down on the list.
Arrow down The key number is the order the button appears in, from left to right, top to bottom. This can be changed by clicking on a line and selecting an arrow to move it up or down on the list.
Edit Click on a line and select this button to open the POS Menu Button View.
Add Click on a line and select this button to open the POS Menu Button View.
Delete Click on a line and select this button to delete.
