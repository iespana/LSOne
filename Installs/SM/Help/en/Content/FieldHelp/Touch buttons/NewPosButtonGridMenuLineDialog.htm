New POS Menu Line Dialog

Store setup > Look and feel > POS button grid menus > Highlight a POS button grid menu line > Select Add button in Menu buttons section

What is the dialog used for?

  • Create and configure a new button grid menu line. Note that it is only possible to add new lines if there is space in the button grid menu. For example, if there are 2 rows and 5 columns available, and there are only 9 lines in the menu buttons, there is one space available and the add button will be available.
Fields & buttons Description
Description Enter a description for the line. This will be shown in the list view.
Operation

Select the operation you want the line to set to the button.

Most of the operations state their parameters clearly in their name. There are three menus, however, that allow you to create layered button grid menus. When you assign a line one of these menus, you then have to assign another button grid menu to each line:

  • Open menu: Open menu opens a menu in another button grid. It is particularly useful if you want to call up many menus and save space on the Touch Button Layout whilst keeping the menu options in view.
  • Popup menu: Select a POS menu to assign to the open menu button. When you click on the open menu, the POS menu you assign to it will appear in another, floating button grid.
  • Sub menu: A submenu is a button that calls up a different menu into the button grid. The menu stays on display until you close it with the "home" button that takes you back to the default view.
  • Another important operation to understand is Item Sale: Item sale is a button that needs to be assigned to an item. The item that you assign the button will appear in the POS receipt panel.
Operation When blank operation is selected, this field appears. Choose the operation that you have pre-configured.
Parameter When operations have more than one option available to choose, this field becomes valid. Select the parameter from the list.
Create multiple lines

Select this check box if you want to create more than one button.

Note that this is only available if there is space in the button grid.
Number of lines to create Enter the number of lines that you want to add.
Copy from Copy from existing POS menu line.
