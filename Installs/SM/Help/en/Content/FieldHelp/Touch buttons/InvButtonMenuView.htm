Omni Button Menu View

Sites > Omni > Button menus > Select Edit button

What is the dialog used for?

  • Edit a button menu.
  • Button menus are used on the POS touch button layouts, in the hospitality button setup and mobile inventory app.
Tab/Field buttons Description
ID Displays the ID of the button menu
Description Edit the description of the button menu
Main menu When selected this menu can be selected as a main menu for the inventory app on the OMNI tab in the functionality profile.
Columns For mobile inventory menus this value is always 1.
Rows Enter the number of rows that you want the menu to have.
Look and feel tab
Tab/Field buttons Description
Style Select the button style. You can create new styles in: Store setup > Look and feel > Styles
Edit Select this button to edit the style that has been selected for this button menu.
Font name Displays the font type used on the button.
Edit Select this button to edit the font type, style, script and size on the button. Open a Font dialog.
Font size Displays the font size.
Font bold If the check box is selected, the font style is bold.
Font italic If the check box is selected, the font style is italic.
Font charset  
Font color Edit the font color by clicking on the box.
Back color Edit the background color of the button by clicking on the box.
Back color 2 Edit the second background color of the button by clicking on the box.
Gradient mode Decide the gradient mode by selecting one of the options on the list.
Shape

Decide the shape of the button by selecting one of the options on the list:

  • Rectangle
  • Round rectangle
  • Triangle
  • Hexagon
  • Ellipse
  • Windows 3D
Menu buttons tab
Tab/Field buttons Description
Key No The key number is the order the button appears in, from left to right, top to bottom. This can be changed by clicking on a line and selecting an arrow to move it up or down on the list. You can then preview the changes to the button grid menu by selecting the Preview button.
Description Displays the description of the button.
Operation Displays the operation performed on the inventory app when this button is used.
Parameter Displays the parameter of the operation (if operation has more than one possibility).
Style Not applicable for the Mobile inventory app buttons
Arrow up The key number is the order the button appears in, from left to right, top to bottom. This can be changed by clicking on a line and selecting an arrow to move it up or down on the list.
Arrow down The key number is the order the button appears in, from left to right, top to bottom. This can be changed by clicking on a line and selecting an arrow to move it up or down on the list.
Edit Click on a line and select this button to open the Menu Button View.
Add Click on a line and select this button to open the Menu Button View.
Delete Click on a line and select this button to delete.
