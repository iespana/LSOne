Gift Card ID View

Retail > Gift cards/Credit Memos > Gift cards > Highlight Line > Select Edit button

What is the dialog/view used for?

  • Edit a gift card.
Tab/Field name Description
ID The gift card ID number.
Balance The gift card balance.
Currency The gift card currency.
Active If the check box is selected then the gift card is available.
Refillable If the check box is selected then the gift card is available.
Usage log
Fields & buttons Description
Store The store to which the gift card was used/created on.
Terminal The terminal that the gift card was used/created on.
Transaction The transaction number on the POS.
Staff The staff member who allowed the transaction. If this does not appear, right click on the line.
Date The date of the transaction.
Amount

The amount on the gift card.

  • When the operation is "Add to gift card" it is the amount added only.
  • "Take from gift card" shows the amount before the gift card was used.
Operation

The following appear on the gift card line to log the actions that have occured:

  • “Activate”/”Deactivate” appears when the gift card was activated or deactivated.
  • “Add to gift card” appears on a line where an amount has been added to the gift card.
  • “Create” when the gift card was created.
  • “Take from gift card” when the gift card was used.
Actions
Tab/Field name Description
Deactivate/Activate Select Deactivate to deactivate the gift card so that it is no longer valid for use. Select Activate to activate the gift card so that it is valid for use. Note that the option is different depending on whether the Gift card is active or inactive.
Add to gift card... Note that this option is only available id the Gift card is refillable. Clicking on this link opens a window where you can enter an amount to add to the selected gift card.
Related
Tab/Field name Description
View gift cards Select View gift cards to go to the Gift cards view.
Other
Tab/Field name Description
Page Arrows Use the arrows to navigate through the gift cards. The numbers 1 out of 5, for example, indicate that you are viewing the first gift card on the list out of a total of five.
