Site Manager Options dialog

Tools tab> Site Manager > Options

What is the dialog/view used for?

  • This dialog is where you can configure the Site Manager, its localization and extensions.
Configurations
Fields & buttons Description
Localization  
Store Determines if the site manager is at head office level or in a store level. When site manager is on store level, most number sequences will automatically add the store ID as prefix and the inventory functionality will be limited to that store.
Edit Use this button to select the store for the site manager.
Current location

Current location has the single function of enabling users to set the current database as a particular location, thus enabling reuse of Scheduler jobs set to use the current location. Should that be selected the location set as the current location will be used when the job is run.
Help  
Help source You can use the online help or the local help which is a file that comes with the system.
Check for update button If using the local help, you can check if there is a new version of the file and update it.
Site Manager hardware profile  
Hardware profile Select the hardware profile to be used by the site manager. Used mainly if you are swiping a card to register its number. If not selected, you will have to remove the MSR characters manually.
Localization
Fields & buttons Description
User interface  
Language Choose the language you want the Site Manager to be displayed.
Conventions  
Name Choose order you want the name to be displayed / printed.
Address Choose the format you want the address to be displayed / printed.
