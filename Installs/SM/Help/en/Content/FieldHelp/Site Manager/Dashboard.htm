Dashboard

What is the view used for?

  • The dashboard displays stickers with various information for the user
  • The dashboard stickers can include shortcut links to various parts of the system
  • The user can control which stickers are visible

How to navigate to the dashboard?

In the toolbar ribbon select General > Navigation > Dashboard

 

Controls

Fields & buttons Description
Back - When all the available stickers on the dashboard cannot be displayed at once this button can be used to scroll to previous page of stickers.
Forward - When all the available stickers on the dashboard cannot be displayed at once this button can be used to scroll to next page of stickers.
Refresh - Press this button to refresh the information on all the stickers.
Configuration - Opens the Dashboard items dialog where you can select which stickers are visible on your dashboard.
