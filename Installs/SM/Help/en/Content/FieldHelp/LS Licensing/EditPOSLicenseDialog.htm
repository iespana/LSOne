Edit POS License Dialog

Security > LS License > POS License > Edit a license for a terminal

What is the dialog used for?

  • If a license for a POS has been obtained in some other way than with the Site Manager the License can be entered/edited here.
Fields & buttons Description
Store Displays which store the terminal the license is valid for is in.
Terminal Displays which terminal the license is valid for.
Password Edit/Enter the license password.
