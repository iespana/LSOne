POS Licensing View

Site Manager > Security > LS Licensing > POS Licensing

What is the view used for?

  • To collect licenses for one or all terminals connected to the system.
  • Note: The POS has to connect to the database for the volume number to be registered in the database. If you have an offline setup (that is terminals have separate databases and data is replicated) then the License info from POS job has to be run from terminal to Site Manager database.
List of terminals

A list of all terminal machines that have been connected to the database.

Fields & buttons Description
Store Displays in which store the terminal is in.
Terminal Displays terminal ID and description the license is for.
Password Displays License password
Collection date Displays the date when the license was collected.
Error
  • Displays false if no error has occurred when getting the license then the line is green.
  • Displays true if an error occurred when getting the license then the line is red. Select the line then the error text is displayed in the lower pane.
Button:
Get all licenses		 Click on this button to collect licenses for all terminals that have been connected to the database.
Button:
Get one license		 Click on this button to get a license for the selected terminal
Button:
Cancel one license		 Click on this button to cancel the license for the selected terminal. Then this license can be collected again later.
Edit Opens the Edit POS license dialog to edit a license password for a terminal. This can be used when the license has been obtained in another way than through the Site Manager.
Delete Deletes the terminal volume number line so it seems that the terminal has never been connected to the database.
Additional information

Displays the volume number for the selected terminal and possible error messages
