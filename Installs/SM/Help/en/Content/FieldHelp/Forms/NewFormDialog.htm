New Form Layout Dialog

Store Setup > Forms and labels > Form layout > Select Add button

What is the dialog used for?

  • Use this dialog to create a new form layout.
Fields & buttons Description
Type

Choose from the drop down list what type of form you would like to create. The type refers to where in the system the form is triggered to be printed. The list shows all the available form types, both system types that come with the system and the user defined types created by the user.

Description Enter a description for your form.
Copy from

Choose from the drop down list an already created form layout shown with the description name. The new form layout will have the same setup as the chosen one.

Is slip Select this check box if the form should be printed on the slip printer.
Print behavior

Select the print behavior of the form:

  • Always print - A form with this selection always gets printed.
  • Never print/Not print - A form with this selection never gets printed.
  • Prompt user/Ask customer - If a form has ask customer selected the LS One POS asks the POS user first
Prompt If Prompt user has been selected in the previous field, type in the text for the prompt. This field is only enabled when prompt user has been selected as print behavior.
Number of columns per line The number of columns of the form. Each column is equivalent to one character.
