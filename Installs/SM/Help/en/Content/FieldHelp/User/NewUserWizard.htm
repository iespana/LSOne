New User Wizard

Security > Users > New User

What is the dialog used for?

  • This dialog is used to create a new user.

To create a new active directory user, it is better to first configure the security login. If the active directory is enabled then you are asked to select a type of user to be created.

Select type of user to be created
Fields & buttons Description
Standalone User Select this option to create a user that is not connected to a Windows domain.

Map an existing domain

user from Active Directory

 Select this option to create a user that uses an existing Windows domain login and password.
Standalone User
Fields & buttons Description
Name Enter a name for the user.
Email Enter an email address for the user.
Login name Enter a login name for the user.
Password Enter a password.
Confirm password Re-enter the password to confirm that it is typed in correctly.
Map an existing domain user from Active Directory
Fields & buttons Description
Name Enter the (existing Windows) name of the user.
Login name Enter the (existing Windows) login name of the user.
Email Enter an email address for the user.
Select roles for the new user
Fields & buttons Description
Copy existing user The user can create a new user with the same properties as the selected existing user. When copying user settings from existing user, permissions greater than the user creating the new user has are not added.
User groups Select which group the new user should belong to, if his settings are not being copied from another user.
User profile Select which user profile the new user should belong to.
Add Add a new user profile. Clicking on this link opens New user profile dialog.
