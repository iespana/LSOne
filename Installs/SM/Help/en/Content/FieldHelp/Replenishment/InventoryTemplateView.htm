Inventory Template View

Inventory > General > Templates > Select inventory template

What is the view used for?

  • Inventory templates contain item filters and settings for stock counting journals, transfer orders and purchase order worksheets.
  • A purchase order template is created to use the item replenishment settings on purchase orders.
  • A stock counting template is used to create stock counting journals preset from a filter of items. Stock counting templates are used in the mobile inventory app.
  • A transfer order template is used to control how items are added to the transfer order how the transfer order can be received.
Fields & buttons Description
ID Displays the inventory template ID.
Description Displays the inventory template description.
All stores check box Select this check box to apply the selected inventory template to all stores.
Store The store that the selected inventory template applies to.
Item filter tab

Create a filter for the template that will be used to populate the worksheet automatically.

Item filter

Retail items can be searched in a filtered search. One filtering line is added for each type of filter. You can even save your preferred filtering settings and they will be available when you open the Retail item view.

Note: The item must have a Unit and a Default vendor to be added to the inventory template

Fields & buttons Description
Retail group

Select a retail group to create a filter for all items in the group
Retail department Select a retail department to create a filter for all items in the department
Special group Select a special group to create a filter for all items in the special group
Vendor Select a vendor to create a filter for all items from this vendor
Begins with This filed is only visible for types that allow free text search.
Search field Enter the search criteria. For description and bar code you can enter a free text but for all the other types you select one group, vendor or such.
If there are more than one line press this button to delete a line from the filter.
Press this button to add criteria to the filter

The following actions are available from a drop down menu when this button is pressed.

  • Reset: Select this to reset the item filtering
  • Toggle button locations: Select this to move the and buttons to the left side of each line.
Button:
Preview		 Press this button or hit enter button to preview the list of items in the filter.

List of items

Fields Description

General tab

All settings that apply to all types of inventory templates are in the general tab.

Fields & buttons Description
Type Displays the type of inventory template.
Unit selection

Select which unit is set by default:

  • Inventory unit
  • Purchase unit
  • Sales unit
Change vendor in line Controls if the user can change the vendor of each line in the worksheet/journal.
Change unit in line Controls if the user can change the unit of each line in the worksheet/journal.
Display barcode Select the check box to display the item barcode in each line.
Purchase worksheet tab

This tab contains all settings that are specific to purchase order templates.

Fields & buttons Description
Calculate suggested quantity If true, then when each line is added to the worksheet the suggested quantity will be calculated and displayed.
Set quantity to suggested quantity If true, then the quantity column should automatically get the same value as the suggested quantity column gets.
Add lines with zero suggested quantity Select this option to display item lines which have not fallen below the reorderpoint and therefore have the suggested quantity: 0.
Display reorder point A column should be displayed in the worksheet with the reorder point information.
Display maximum inventory A column should be displayed in the worksheet with the max inventory information.
Stock counting tab

This tab contains all settings that are specific to stock counting templates.

Fields & buttons Description
Type of entering

Select the type of entering:

  • Add to quantity: Select this option to sum up all lines for the same item
  • Add line: Select this option to keep each line that has been entered for the same item.
Omni inventory All these settings only apply to the mobile inventory app
Quantity method

Select one of the following methods for adding lines to the stock counting worksheet.

  • Ask - When items are added to the stock counting worksheet the app asks for the counted quantity of the item.
  • Quick scan - When an item is added to the stock counting worksheet the app used the default quantity set in next field. Then the quantity can be edited.
Default quantity Enter a number to be the default quantity of the item lines in the stock counting journal. The lines can be edited. This default quantity is only used when using the quantity method Quick scan.
Area Displays the name of the Area section the template is fore. Each area can have multiple area lines and each stock counting journal can select an area which is being counted.
Edit Press this button to edit the name of the Area
Area lines Displays all the Areas this stock counting template is for.
Button:
Copy from		 Press this button to copy areas from other stock counting templates. Opens the Copy area dialog
Edit Select an area line and press this button to edit the area line. Opens the inventory template area dialog.
Add Press this button to add a area line to the stock counting template. Opens the inventory template area dialog.
Delete Press this button to delete an area line from the stock counting template.
