Currencies View

General Setup > Payments > Currencies

What is the view used for?

  • Here, all the currencies that are going to be used by the currency payment type are set up.
  • In the upper pane of the window is a list of the valid currencies and the lower pane includes an exchange rate for the selected currency.
  • Note:
    • The default currency for the company has to be included on this list with an exchange rate of 1,000.
    • The default currency for the company is written in bold letters.
    • The default currency for the Demo Data company is EUR.
    • The companyâ€™s default currency is set up in Store setup > Stores > Company information.
    • Every Store can have an individual default currency in its set up.
Fields & buttons Description
Currency code Displays the currency ID.
Description Displays the currency description.
Is company currency

The company needs to have one currency selected as the company currency. Prices set in the Site Managerare in this currency, unless otherwise stated and will be converted to other currency using the set exchange rate for that currency.

The company currency always has the exchange rate set as 1.

Button:
Edit company currency		 By pressing this button the Company information view is opened where the company currency can be selected and edited.
Edit Highlight a line and select this button to edit the currency. Opens the Currency dialog.
Add Select this button to create a new currency. Opens the Currency dialog.
Delete Highlight a line and select this button to delete the currency.
Exchange rate tab

The exchange rate per currency is set up with a start date and is valid until its next start date. It is very different between setups and, or countries how often the exchange rate is entered, every day, week or month.

You can keep track of the history of the currency rate by inserting each new exchange rate.

Note: Please be aware that it is bad practice to edit the currency instead of creating a new line each time the currency rate changes.

Fields & buttons Description
Start date The start date of this exchange rate.
Exchange rate The official exchange rate. The column shows the currencies that are being converted. For example (EUR -> GBP)
POS exchange rate The exchange rate used on the LS One POS terminal.
Edit Highlight a line and select this button to edit the exchange rate. Opens the Exchange rates dialog.
Add Select this button to create a new exchange rate. Opens the Exchange rates dialog.
Delete Highlight a line and select this button to edit the exchange rate.
Currency units tab

Create currency units for cash.

Fields & buttons Description
Type

Displays the type of denomination, this is either:

  • Coin
  • Bill
Amount Displays an amount that is accepted by the store in this currency.
Add Select this button to create a new currency unit. Opens the Cash denomination dialog.
Delete Highlight a line and select this button to edit the currency unit.
