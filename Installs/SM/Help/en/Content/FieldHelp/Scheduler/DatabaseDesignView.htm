Database Design View

General setup > Replication > Database design

What is the view used for?

  • To manage database designs and link tables.
List of database designs - to the left

A tree view showing all database designs

Fields & buttons Description
Show only where linked tables exist When this is check marked, only table designs having linked tables will be shown.
Show disabled databases

When this option is check marked, the tree view will include database designs that have been marked disabled.

Disabled designs have a gray icon while the others have green.

Database design - to the right

If a database design is highlighted in the list on the left then database design pane is displayed on the right

Fields & buttons Description
Description Displays the description of the database design
Code page An optional code page used in the database design, for example 861
Enabled Uncheck this if you don’t want to be able to set assign this database design to a location.
Locations where used A list of all the distribution locations using this database design
Name Displays the description of the distribution location.
External code Displays the ID of the distribution location.
Table design - to the right

If a table is highlighted in the list on the left then table design pane is displayed on the right

Fields & buttons Description
Table name Displays the name of the database table.
Enabled Unselect this if you don’t want to be able to create a subjob for this table.
Fields A list of all fields in the selected table
Name Displays the name of the database field.
Data type Displays the data type of the database field.
Length Displays the length of the database field.
Enabled Displays if the database field is enabled.

Action

Read fields - The details of the table view cannot be edited. However, you can read (or re-read) the field design of a table by clicking the Read fields in the database design menu.

Linked table- to the right

If a linked database table is highlighted in the list on the left then linked table pane is displayed on the right

Fields & buttons Description
Link to table design Displays the name of the linked table.
From field Displays the name of the field in the linked table.
Link type This setting is not available in this version.
To field Displays the name of the field in the original table.
Filter This setting is not available in this version.
