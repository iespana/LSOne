Loyalty Cards View

Retail > Loyalty > Loyalty Cards

What is the view used for?

  • Go to loyalty cards to view and edit cards that have been created.
  • This is a list view of all of the cards on the system. Search for cards using filters.
Search filtering

Loyalty card searches can be narrowed down using filters. One filtering line is added for each type of filter. You can even save your preferred filtering settings and they will be available when you open the Loyalty Card view.

List of cards
Fields & buttons Description
Card number The card number.
Type (card type)

The card type (how the loyalty card should be used):

  • Card tender: The card can be used for accruing and spending points
  • No tender: The card cannot be used as tender in a POS transaction, but can be used for accruing points and for other kinds of uses such as to access online deals.
  • Contact type: If a customer has multiple cards, this card gathers the points information from all the loyalty cards registered to the customer and allows the customer to use them all together.
  • Blocked: The card is blocked for use by the customer. *This cannot be used for new loyalty cards, only for editing loyalty cards in Edit loyalty card dialog in the Loyalty cards view.
  • Scheme ID: Select a scheme that has already been created.
Customer Customer assigned to card. Assigning a customer to the card issues it.
Scheme Description of the loyalty scheme. All loyalty cards must have a loyalty scheme. A scheme can be thought of as a container for one or more rules that will either increase (accrue) or decrease (spend) the loyalty points on the card. The scheme can have one or more rules within it.
Starting points Number of points on the card when issued.
Issued Date that the card is issued/activated.
Used Number of points used on the card. Used points are displayed with a minus symbol as a prefix (-).
Expired Current points expiry date.
Balance Current balance of points on loyalty card.
