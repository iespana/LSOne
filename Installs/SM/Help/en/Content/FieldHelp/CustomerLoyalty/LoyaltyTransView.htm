Loyalty Transactions View

Retail > Loyalty > Loyalty transactions

What is the view used for?

  • View all loyalty transactions in the Loyalty transactions view.
  • Each transaction on a loyalty card is shown separately in list form.
  • There can be many transactions on the same card, and the card can have the status closed and still be valid for use. This just means that the points in the scheme have either been used or have expired according to the schemeâ€™s points expiration rule.
  • The loyalty card itself does not have an expiration date, but the scheme and scheme rules do.
Search filtering

Loyalty transaction searches can be narrowed down using filters. One filtering line is added for each type of filter. You can even save your preferred filtering settings and they will be available when you open the Loyalty Transactions view.

Loyalty Transactions List

View the loyalty transactions.

Card type/Card status possible combinations:

Issued/Closed:

  • Points have been issued on the loyalty card.
  • All of the points on the card have been used in this transaction or expired since.
  • The card is still valid for accruing/spending.

Issued/Open

  • The card has been issued to a customer or accrued points.

  • There are points remaining on the card.

  • The points are available for spending.

Used/Closed

  • Points were used in this transaction.

  • The card is still valid for accruing/spending.

Expired/Closed

  • The points on the card have expired.
  • The card is not valid for accruing/spending.
Fields & buttons Description
Type

Select one of the following from the list: 

  • Issued: The loyalty points have been issued to a loyalty card or accrued in a transaction.
  • Used: The loyalty points have been used for payment or discount in a transaction.
  • Expired: The loyalty points in this line have expired.
Receipt number Displays the receipt number on the transaction. There can be many receipt numbers for the same loyalty card.
Loyalty points Displays the number of loyalty points accrued or spent in the transaction. If they have been spent, the number's prefix is a (-) symbol.
Remaining points Displays the number of remaining points. This is never a minus number. If 0, the transaction has been used for payment or issued.
Status

The card status can be:

  • Closed: The remaining points are zero.
  • Open: The transaction accrued points on the card.
