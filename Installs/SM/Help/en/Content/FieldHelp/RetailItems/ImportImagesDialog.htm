Import Item Images Dialog

Retail > Retail items > Retail items > Import images link

What is the view/dialog used for?

Import multiple images. There are three ways to import images:

  1. Files in a folder: import multiple images from one folder;
  2. Based on a data file: use a text file to import multiple images from multiple locations
  3. Add a single image through the Retail item view using the Add button.

 

Fields & buttons Description
Images to import In this section you choose if you want to import images from a folder or based on a data file.
Files in a folder Selecting this option allows you to import multiple images with the same name as the item ID of the retail items from a folder on your computer.
Image folder The path of the image folder is displayed here.
Click on this button to open Browse for folder dialog.
Images to import Delete any extra rows
Based on data file

Select one of the three following options from the drop-down list menu:

  • All image files: Both .png and .jpg formatted files are imported.
  • PNG files: Only .png formatted files are imported.
  • JPG files: Only .jpg formatted files are imported.
Data file The path of the data file is displayed here.
Image alteration In this section you can set alterations to the image, resizing it or compressing it.
Set maximum size Select this checkbox to make the Resize image area valid.
Resize image Edit the Maximum pixels on longer edge directly into the box or use the arrows to go up or down one pixel at a time.
Compress images Select this checkbox to make the Use compression area valid.
Use compression Select the percent that you want the image to be compressed by, by selecting a number from the drop-down menu.
Progress In this section you can see the progress of the import.
Progress indicator When the green color fills this area, the import is complete.
Imported images The number of imported images is displayed here.
Failed imports The number of failed imported images is displayed here.
Skipped files The number of files that were skipped during import is displayed here.
