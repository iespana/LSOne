New Item Dialog

Item > Items > Add item

What is the view/dialog used for?

  • To create a new item or header item
  • You can also create a series of new items or header items by selecting the create another check box in the dialog footer
  • Set all the mandatory settings on the item which you can change later in the Retail item ID view.

The new New item dialog is the first dialog that has tabs. See field help under the following sections:

General tab
Fields & buttons Description
ID

Enter an ID for the new item. If no ID is entered a numeric ID is generated for your item.

This field is visible only if manually entered item IDs are allowed.

The allow manually entered item ID setting is found in General setup > System > Administrative options > Store management.
Item type Select the item's type (retail item, service item or variant header).
Description Enter a description/name for the new item. You can enter a maximum of 60 characters, both numbers and letters. This description is typically printed on orders and invoices. Enter it as you want it to appear.
Retail group Select a retail group for the item. If the Retail group has set up an item group and variation group, these fields are automatically filled for this item. The item also inherits the Retail department settings from its Retail group.
Add Opens the New retail group dialog to create a new retail group.
Variation group The Variation group settings are inherited from the Retail group. These settings can be changed for a new item.
Add Opens the Variation group dialog to create a new variation group.
Inventory unit Select the unit of measure the item is purchased in. This unit can differ from the sales unit (the unit the item is sold in), but then the inventory unit and sales unit must have a unit conversion rule set up.
Add Opens the Unit dialog to create a new unit.
Sales unit Select the unit of measure the item is sold in. The inventory unit and sales unit must have a unit conversion rule set up.
Add Opens the Unit dialog to create a new unit.
Item sales tax group Select an item sales tax group for your item.
Price Enter a price for the item. If your Default tax store prices are set to include tax: This is the price including tax. If your default tax store price is set to be without tax then the price entered is the base price of the item without tax.
Bar codes tab
Fields & buttons Description
Bar code setup Select a bar code setup. Note: you must enter a default bar code if you choose a bar code setup.
Default bar code Enter a default bar code. Note: you must select a bar code setup to enter a default bar code.
Dimensions tab
Fields & buttons Description
Dimensions This is a list of all the dimension this item can have. Dimensions are properties of the item such as color, size or style.
Description Displays the name of the dimension
Up Move the selected dimension one step up the list to order the dimensions. This affects how the variant description is formed, that is in what order the dimensions are listed.
Down Move the selected dimension one step down the list to order the dimensions. This affects how the variant description is formed, that is in what order the dimensions are listed.
Edit Select a dimension from the list and press this button to edit the dimension description
Add Press this button to create a new dimension for this item.
Delete Select a dimension from the list and press this button to delete it.
Dimension attribute This is a list of all the attribute the selected dimension from the upper list can have. For an example if dimension color is selected the attributes could be red, green and blue.
Description Displays the name of the dimension attribute
Code Displays the code of the dimension attribute. The code can be used when searching.
Up Move the selected attribute one step up the list to order the attributes.
Down Move the selected attributes one step down the list to order the attributes.
Edit Select an attribute from the list and press this button to edit the attribute description or code
Add Press this button to create a new dimension attribute for this item.
Delete Select a dimension attribute from the list and press this button to delete it.
Variant item descriptions preview Displays an example of a generated description for a variant item.
Variant items tab
Fields & buttons Description

Search filters:

  • Description
  • Selected

Select a filter type for each search filter line.

Check the box to use the search criteria.

 
  • Begins with
  • Contains
  • Begins with: Selecting this option allows you to search for the beginning of the customer description, ID or Account number. (This is the default setting)
  • Contains: Selecting this option allows you to search for the number or letters within the customer description, ID or Account number.

This field is only visible for types that allow free text search. That is description, variant and bar code.

If there are more than one line press this button to delete a search criteria.

You can also unselect the checkbox to exclude a criteria from the search.

Press this button to add search criteria and narrow the filter.

The following actions are available from a drop down menu when this button is pressed.

  • Save as default layout: The search lines you have selected will be saved to the view and opened when this view is opened.
  • Load default layout: If you have edited the search settings since you last opened the Retail items view select this option to reload the saved layout.
  • Reset: Select this option to reset the layout to be as displayed in image above.
  • Toggle button locations: Select this option to set the and buttons in front of the filter type.
Button:
Filter		 Press this button or hit enter button on keyboard to filter the list of items according to your settings.
Button:
Select all		 Press this button to select all dimension combinations.
Button:
Select none		 Press this button to deselect all dimension combinations.
Fields & buttons Description
Create another Check this box to keep the New item dialog open and ready to create the next item after you press OK. You can always click the cancel button to discontinue.
