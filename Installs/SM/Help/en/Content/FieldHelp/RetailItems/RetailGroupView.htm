Retail Group View

Retail > Retail items > Retail groups > Edit a retail group

What is the view/dialog used for?

  • To edit the settings for a retail group
  • To add infocodes to the retail group
Header fields Description
Retail group Contains the ID of the retail group.
Description This is the description of the retail group.
General tab

A list of settings set on the retail group, like the retail groupâ€˜s place in the item hierarchy and some settings that new items in the retail group will inherit.

Fields & buttons Description
Retail department Select the retail department the retail group belongs to.
Edit Opens the retail departments view. The selected retail department will also be selected in the retail departments view.
Item sales tax group Select an item sales tax group that all new retail items will inherit when they are created. These settings can be changed for each new item.
Variation group Select a variation group that all new retail items will inherit when they are created. These settings can be changed for each new item.
Edit Opens the Variation groups view where you can edit the variation combinations for the group.
Size group Depending on the size, color, style combination on the selected variation group select a size group that the retail item will inherit when created. These settings can be changed for each new item.
Color group Depending on the size, color, style combination on the selected variation group select a color group that the retail item will inherit when created. These settings can be changed for each new item.
Style group Depending on the size, color, style combination on the selected variation group select a style group that the retail item will inherit when created. These settings can be changed for each new item.
Infocodes tab

A list of infocodes that are activated for all items in the retail group.

Fields & buttons Description
Edit Opens the New link dialog to edit the selected infocode linking settings.
Add Opens the New link dialog to select an infocode to link to the retail group.
Delete Deletes the selected infocode from the list.
Cross selling tab

A list of cross selling infocodes that are activated for all items in the retail group.

Fields & buttons Description
Edit Opens the New link dialog to edit the cross selling infocode linking settings.
Add Opens the New link dialog to select a cross selling infocode to link to the retail group
Delete Deletes the selected infocode from the list.
Item modifiers tab

A list of item modifiers that are activated for all items in the retail group.

Fields & buttons Description
Edit Opens the New link dialog to edit the item modifier linking settings.
Add Opens the New link dialog to select an item modifier to link to the retail group.
Delete Deletes the selected infocode from the list.

Actions

  • Infocodes - Select this link to open the infocodes view to edit or add an infocode.
