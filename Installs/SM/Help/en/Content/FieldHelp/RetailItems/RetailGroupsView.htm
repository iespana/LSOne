Retail Groups View

Retail > Retail items > Retail groups

What is the view/dialog used for?

  • This view gives an overview of the retail groups and also which items are included in which retail group.
Search filtering

Retail items can be searched in a filtered search. One filtering line is added for each type of filter. You can even save your preferred filtering settings and they will be available when you open the Retail item view.

Fields & buttons Description
  • Description
  • Style group
  • Color group
  • Size group
  • Validation period
  • Variant group
  • Tax group
  • Retail department

Select a filter type for each search filter line.

Check the box to use the search criteria.

 
  • Begins with
  • Contains

This filed is only visible for types that allow free text search. That is description and bar code.

Search field Enter the search criteria. For description and bar code you can enter a free text but for all the other types you select one group, vendor or such.
If there are more than one line press this button to delete a search criteria.
Press this button to add search criteria

The following actions are available from a drop down menu when this button is pressed.

  • Save as default layout: The search lines you have selected will be saved to the view and opened when this view is opened.
  • Load default layout: If you have edited the search settings since you last opened the Retail items view select this option to reload the saved layout.
  • Reset: Select this option to reset the layout to be as displayed in image above.
  • Toggle button locations: Select this option to set the and buttons in front of the filter type.
Button:
Search		 Press this button or hit enter button on keyboard to search for an item with the filters set.
List of retail groups

A list of all existing retail groups.

Fields & buttons Description
Retail group Contains the ID of the retail group.
Description Contains the description of the retail group.
Retail department The description of the Retail department the group belongs to.
Edit Opens the Retail group view where you can edit the group.
Add Opens the New retail group dialog.
Delete Deletes the selected retail group from the list.
Items in group

A list of all items in the retail group selected in the list of retail groups.

Fields & buttons Description
ID Contains the ID of the retail groups in the department.
Description Contains the description of the retail groups in the department.
Back Select this button to scroll back through the list of retail items. This button is only enabled if there are more than 500 items in the retail group.
Next Select this button to scroll forwards through the list of retail items. This button is only enabled if there are more than 500 items in the retail group.
Edit Opens the Item view for the selected item from the Items in group list.
Add Opens the Item search dialog where you can select new items to add to the retail group.
Delete Deletes the selected item from the retail group.
