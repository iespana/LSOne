Retail Items View

Retail > Retail items > Retail items

What is the view/dialog used for?

  • View all of the retail and service items available.
  • Search for groups, departments and individual items using the search filter.
  • Scroll the item pages with the forward/back buttons.
  • Import multiple images to match items through the image import dialog.
  • Note: you can choose which column the items are ordered by, by clicking on the column header, it will automatically choose an upwards pointing arrow that orders 0-10 and A-Z, reverse the order by clicking again and making sure that the green arrow points downwards.

Different types of items

The items view displays all sorts of items:

  • retail items
  • service items
  • variant headers
  • variant items

Multiple items can be selected and some of their settings can be edited all at once. This is called bulk editing.

Search filtering

Retail items can be searched in a filtered search. One filtering line is added for each type of filter. You can even save your preferred filtering settings and they will be available when you open the Retail item view.

Information on filtering search by attributes click here.
  • Begins with
  • Contains
  • Begins with: Selecting this option allows you to search for the beginning of the item description or ID. (This is the default setting) .
  • Contains: Selecting this option allows you to search for the number or letters within the item description or ID.

This field is only visible for types that allow free text search. That is description, variant and bar code.

  • All types
  • Items
  • Service items
  • All types: Search does not exclude any type of item.
  • Items: Search only displays items that are not service items.
  • Service items: Search displays only items that are service items.
Search field Enter the search criteria. For description and bar code you can enter a free text but for all the other types you select one group, vendor or such.

If there are more than one line press this button to delete a search criteria.

You can also unselect the check box to exclude a criteria from the search.

Press this button to add search criteria and narrow the filter.

The following actions are available from a drop down menu when this button is pressed.

  • Save as default layout: The search lines you have selected will be saved to the view and opened when this view is opened.
  • Load default layout: If you have edited the search settings since you last opened the Retail items view select this option to reload the saved layout.
  • Reset: Select this option to reset the layout to be as displayed in image above.
  • Toggle button locations: Select this option to set the and buttons in front of the filter type.

List of items
Fields & buttons Description
ID Displays the item ID.
Description Displays the item description.
Variant This field is only visible if variant items exist in the item list. If there are variant items in the list this column will display their variant description that is the dimension attributes of the variant item.
Retail group Displays the item retail group.
Retail department Displays the retail department.
Tax group Displays the tax group.
Price Displays the item price including tax, if any.
Back Select this button to scroll back through the list of retail items. This button is only enabled if there exist more than 500 items.
Next Select this button to scroll forwards through the list of retail items. This button is only enabled if there exist more than 500 items.
Button:
Export

Select one of the following options:

  • List to Excel template... - Export all items to an Excel document where the item settings can be modified before importing again. See Excel import for more details.
  • List to Excel replenishment template... - Export all items to an Excel document where the items replenishment settings can be modified before importing again. See Item view - Replenishment tab for more information on these settings.
  • List to Excel stock counting template... - Export all items to an Excel document where the item stock counting settings can be modified before importing again. See Excel import for more details.
Button:
Import

Select one of the following options:

  • Images - Select this option to import multiple images at once to items. This opens the Import item images dialog.
  • From Excel document... - Browse to the Excel document that you want to import. The Excel document has to be one of the templates that the Site Manager has created.
Edit Highlight an item on the list and select this button to edit it. This opens the item’s Retail item ID view.
Add Select this button to add a new item. This opens the New item dialog.
Delete

Highlight an item on the list and select this button to delete it.

All items can be deleted as nothing is actually deleted from the data, items are only marked as deleted and can be undeleted. To undelete an item select the attribute search filter and select deleted. Then all deleted items will be displayed in the search results.

Actions

  • Label printing - Opens the Label printing dialog from where price or shelf labels can be printed. Before printing labels printing templates have to be created in the Store setup > Forms and labels > Item label templates.
  • Import images - Select this link to import multiple images at once to items. This opens the Import item images dialog.
