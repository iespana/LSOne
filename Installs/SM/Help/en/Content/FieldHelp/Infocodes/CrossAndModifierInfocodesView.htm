Cross Selling Infocodes Groups/Item Modifier Infocodes Groups View

Retail > Infocodes > Cross selling infocode groups/ Item modifier infocode groups

What is the view used for?

  • Create or edit Cross selling infocode groups and Item modifier infocode groups.

Fields & buttons Description
Infocode group type

Choose a group type from the list:

  • All types
  • Item modifiers
  • Cross selling
List of all cross selling/item modifier groups

A list of all existing retail groups.

Fields & buttons Description
Group ID Displays the group’s ID.
List type

Displays the group’s list type:

  • All types
  • Item modifiers
  • Cross selling
Description Displays the group’s description.
Header text Displays the group’s header text.
Triggering Displays whether the group is triggered automatically or manually.
OK pressed action
  • Displays the action performed when OK is pressed on the POS.
  • Either Jump to next infocode in line or Jump to next required infocode.
Minimum selection Displays the minimum triggers needed to trigger the infocode.
Maximum selection Displays the maximum triggers needed to trigger the infocode.
Linked infocode Displays the linked infocode, if there is one.
Edit Highlight a line and select the Edit button to edit the group.
Add Select Add to add a group.
Delete Highlight a line and select Delete to delete it.
Infocode selection lists within a group

A list of all existing infocodes in the selected group.

Fields & buttons Description
Trigger code Displays the Trigger code.
List type Displays the list type.
Description Displays the infocode description.
Infocode prompt Displays the infocode prompt.
Add Select the Add button to create a new infocode selection list within the group.
Delete Highlight a line and select Delete to delete it.
