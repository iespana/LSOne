Infocode View

Retail > Infocodes > Infocodes > Highlight Infocode Line and select Edit button/Select Add button and confirm description

What is the view/dialog used for?

  • The view is used to create and edit an infocode.
General tab
Fields & buttons Description
Code The code automatically generated by the system for the infocode.
Description The infocode description. A text box that you can edit.
Prompt The infocode prompt. A text box that you can edit.
Type

Select the infocode type:

  • Numeric input
  • Text input
  • Date input
  • Item selection
  • Customer selection
  • Age limit
  • Selection list
  • Selection buttons
Minimum value Enter the minimum value needed for the infocode to appear. (Appears with Numeric input and Age limit).
Maximum value Enter the maximum value needed for the infocode to appear. (Appears with Numeric input).
Minimum length Enter the minimum length needed for the infocode to appear. (Appears with Text input).
Maximum length Enter the maximum length needed for the infocode to appear. (Appears with Text input).
Minimum selection Enter the minimum selection needed for the infocode to appear. (Appears with Selection list and Selection buttons).
Maximum selection Enter the maximum selection needed for the infocode to appear. (Appears with Selection list and Selection buttons).
Triggering Choose whether the infocode is triggered automatically or on request.
Input required Select this check box if input is required by the infocode.
Once per transaction Select this check box if the infocode can only appear once per transaction.
Linked infocode Select this check box to link the infocode to another infocode that you can select from the drop down list.
Random factor % Type in the random factor percentage if you want the infocode to appear 20% of the time, for example, type 20 in the text box.
Printing tab
Fields & buttons Description
Print prompt on receipt Select this check box if you want to print the infocode prompt on receipt.
Print answer on receipt Select this check box if you want to print the answer given to the infocode on the receipt.
Print extra information on receipt Select this check box if you want extra information to be printed on the receipt.
