Find Receipt View

Retail > Transactions > Search for receipt

What is the view used for?

  • Use this view to view and search for receipts.

  • The top part of the view contains filters that can help you narrow down your search to more specific values.

  • You can search by date and time, receipt number, employee, store, and/or terminal.

Find Receipt View
Fields & buttons Description
Date range Select this check box to search for a receipt from the date that you select on the calendar. You have the option of selecting the time of the day that the system should start searching.
to Select this check box to search for a receipt to the date that you select on the calendar. You have the option of selecting the time of the day that the system should stop searching.
Receipt number Search for a receipt by entering a number that begins with or contains the inserted number.
Employee Search for a receipt by selecting the employee on the transaction from the drop-down list.
Store Search from a receipt by selecting the store on the transaction from the drop-down list.
Terminal Search for a receipt by selecting the terminal on the transaction from the drop-down list.
Button:
Search		 Select this button to search for a receipt with the filters that you have selected.
Time Displays the receipt’s date and time.
Receipt number Displays the receipt’s number.
Employee id Displays the employee on the receipt.
Terminal id Displays the terminal on the receipt.
Store Displays the store on the receipt.
Total amount Displays the transaction's total amount.
Button:
View		 Highlight a receipt and select this button to view it. Opens the Receipt view.
Select View button

This view displays receipts. Select options from the toolbar.

• The receipt view allows you to edit, externally save and print the receipts.

• Changing the background color might be useful for indicating different types of receipts.

• Saving the receipts to an external hard-drive might be useful if you need to keep back-ups.

• You can save/export the receipt in many different forms, from PDF to image files.

• Find a particular element on the receipt by using the search option, saving you time.

Fields & buttons Description
View Select this option to control the view of the receipt. If the options have a tick next to them, it means that they have been selected.
Page Layout

Facing: Select this option to print pages face each other.

Continuous: Select this option to print pages in continuous form.
Toolbar Select this option to hide the toolbar. If the toolbar has a tick next to it, it is on display.
Background Select this option to change the color of the receipt’s background.
Color

  • Web

Search Select this button to open the Find dialog.

Find Next Select this button to find the next part of the same parameter entered into the Find what field. When you select this button the area where the parameter entered is highlighted on the receipt.
Match case Select this check box to match a case.
Match whole word Select this check box to match a whole word.
Search up Select this check box to search up.
Close Select this button to close the dialog.
Hand tool Select this option to change the cursor to a hand tool that you can use to drag the receipt up or down with.
Open Open a file from the hard drive.
Save Save a file to the hard drive.
Print as Open the print dialog to select the printer.

Page setup Open the page setup dialog.

Zoom out Select this button to zoom out of the view.
Zoom in Select this button to zoom into the view.
Arrows Select the arrows to browse the receipt.
Export document Select this button to export the document in many formats, including PDF, HTML and image files.
Send via e-mail Select this button to send the document in many formats, including PDF, HTML and image files.
