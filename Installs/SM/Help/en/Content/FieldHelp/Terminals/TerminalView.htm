Terminal View

Store setup > Terminals > Terminals > Edit a terminal

What is the view/dialog used for?

  • Enter/edit the settings for a POS terminal.
Fields & buttons Description
ID The ID of the terminal.
Description Enter or edit the description of the terminal here.
Store Select the store from the drop-down list.
Edit Opens the Store view to edit the store.
Hardware profile

Select the hardware profile from the drop-down list.
Edit Opens the Hardware profiles view to edit the hardware profile.
Visual profile

Select the visual profile from the drop-down list.
Edit Opens the Visual profiles view to edit the visual profile.
Functionality profile

Select the functionality profile from the drop-down list.
Edit Opens the Functionality profiles view to edit the functionality profile.
Touch buttons

Select the touch buttons profile from the drop-down list.
Edit Opens the Touch button layouts view to edit the touch button profile.
Settings tab
Fields & buttons Description
Automatic logout Select the check box to enable the automatic logout from the POS.
Minutes Enter the number of minutes before the automatic logout takes place.
Automatic lock Select the check box to enable the automatic lock of the POS.
Minutes Enter the number of minutes before the automatic lock takes place.
Open drawer at logout Select the check box to open the drawer at logout.
Log out after each transaction Select the check box to log out of the POS after each transaction.
Terminal is independent Select the check box to keep the terminal independent from others.
Receipt number series Select the receipt number series from the drop-down menu.
Form info 1 Enter additional information specific to the store and should be printed on a form. The Terminal info 1 field has to be added to the form in the form layout editor.
Form info 2 Enter additional information specific to the store and should be printed on a form. The Terminal info 2 field has to be added to the form in the form layout editor.
Form info 3 Enter additional information specific to the store and should be printed on a form. The Terminal info 3 field has to be added to the form in the form layout editor.
Form info 4 Enter additional information specific to the store and should be printed on a form. The Terminal info 4 field has to be added to the form in the form layout editor.
EFT settings
Fields & buttons Description
IP Address

Enter the IP address of the EFT device connected to the terminal.

Note that whether this field is used by the EFT implementation is implementation specific.
EFT cash register ID Enter/edit the unique id for this register. The use of this field is EFT implementation specific.
EFT store ID Enter/edit the store id for this register. The use of this field is EFT implementation specific.
Custom field 1 This field is EFT implementation specific. Consult the document for the EFT implementation.
Custom field 2 This field is EFT implementation specific. Consult the document for the EFT implementation.
Customer display
Fields & buttons Description
Primary text Use the text box to enter the first line of text that is shown on the customer display.
Secondary text Use the text box to enter the second line of text that is shown on the customer display.
Hospitality
Fields & buttons Description
Switch user when entering POS If this option is selected the POS will prompt the user to log in after a table has been selected to sell on. Then the POS can be left logged into the table layout but you will have to log in to be able to sell on a table.
Hospitality types filter A list of all existing hospitality types. Select the hospitality types that should be accessible on this terminal.
Select all Select this button to select all of the Hospitality types shown.
Select none Select this button to deselect all of the Hospitality types shown.
Site service
Fields & buttons Description
Headoffice Site Service profile Select the Site Server profile for the headoffice from the drop-down list.
Edit

Select the edit button to edit or add a Site Service profile.
Store Site Service profile Select the Site Server profile for the store from the drop-down list.
Edit

Select the edit button to edit or add a Site Service profile.
Suspended transaction
Fields & buttons Description
Warn on statement posting

Select one of the following options from the drop-down list to determine if performing end of day operations are allowed when there are suspended transactions in the system:

  • Store default
  • Yes
  • No
End of day
Fields & buttons Description
Include terminal in statement

Check this field if you want to include the terminal in statement
Allow posting Choose Always or Only if last transaction is "End of day"
