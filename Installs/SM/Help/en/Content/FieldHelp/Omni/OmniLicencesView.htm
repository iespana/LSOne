Omni Licenses

Sites > Omni > Omni Licenses

What is the view/dialog used for?

  • An overview of all mobile devices that have been connected to LS One
  • Get the device ID of the mobile devices to send to licensing to order a license
  • Enter the license ID you get from licensing
Fields & buttons Description
Application ID

Defines which of the omni apps was used to connect to LS One

  • POS - the LS Omni Mobile POS
  • INV - the LS Omni Inventory app
Device ID Displays the ID of the mobile device that has connected to LS One.
Terminal ID Displays the ID of the terminal this device is connecting as.
Store ID Displays the ID of the store the Omni device is connecting to.
License key Displays the license key for each device.
Edit

Press this button to edit the Omni license line. Opens the Edit Omni License Dialog

In this dialog the Device ID can be copied to be able to send it to with the license order and the license key can be paste into the license field.

Delete Press this button to delete an omni license line. If devices are no longer in use you can clean this list by deleting the devices that are not used.
