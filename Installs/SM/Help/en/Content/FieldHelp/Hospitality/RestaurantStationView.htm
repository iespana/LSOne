Printing Station ID View

General setup > Hospitality > Printing station > Select printing station

What is the view used for?

  • Each Restaurant station (bar, wine cellar, and kitchen) in the restaurant needs to have its own configuration.
  • Apply the settings in this view.
Fields & buttons Description
ID Displays the station ID.
Station name Enter/edit the station name.
Station type

Controls how the station prints the items.

  • OPOS printer: Will use the LS Station Printing service to print the items on a remote OPOS printer.
  • Windows printer: Will use a network printer to print the items.
  • Hardware profile printer: Will use the local OPOS printer configured in the terminalâ€™s hardware profile to print the items.
  • Note:  If the station printing should use the same printer as the POS then this option should be selected. For example, when there is only one printer and everything prints through that printer.
Font name Displays font name. Only active with Windows printer.
Edit Opens Font dialog. Change font by clicking here.
Windows printer The network path to the printer. Is only active if the Station type is set to Windows Printer.
Station printing host The IP address and port where the LS Station Printing service is. It is only active if OPOS printer is selected in Station type.
Edit Click on this button to open the Printing station hosts dialog.
OPOS device name Select the OPOS device name.
OPOS vertical font size Select numbers 1-3.
OPOS horizontal font size Select numbers 1-3.
OPOS max characters Enter/edit the max characters available.
View all restaurant stations Click on this link to go to Printing stations view.
