Dining Table Layout View

General setup > Hospitality > Hospitality types > Select hospitality type and click on Edit button > Select Table button tab > Dining table layout id > Select Edit button

What is the view used for?

  • View and edit the dining table layout on the POS.
Fields & buttons Description
Layout ID Displays the layout ID.
Description Edit the description in this text box.
Number of dining tables Displays the number of dining tables on the list in the view below.
Starting table number Displays the starting table number on the table view for this hospitality type.
Ending table number Displays the ending table number on the table view for this hospitality type.
Dining table rows Edit the number of dining table rows displayed on the table button view for the POS. This automatically sets to fit the right number of dining tables.
Dining table columns Edit the number of dining table columns displayed on the table button view for the POS. This automatically sets to fit the right number of dining tables.
Table no. Displays the table’s number.
Default description Displays the table’s default description.
Description on button Displays the description on the button.
Non smoking Displays “Yes” if the hospitality type is set to be non smoking and “No” if it is set to smoking.
Availability Displays “Available” if the table is available for use on the POS and “Not available” if the table is not available on the POS.
Edit Highlight a line and select this button to edit the Dining table. Opens the Edit restaurant dining table dialog.
Add Select this button to add a new dining table. Opens the Add restaurant dining tables dialog.
Delete Highlight a line and select this button to delete it.
