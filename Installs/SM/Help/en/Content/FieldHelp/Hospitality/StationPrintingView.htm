Station Printing ID View

General setup > Hospitality > Station printing > Select Edit button

What is the dialog used for?

  • Edit a printing station.
Fields & buttons Description
ID Displays the ID of the printing station.
Description Edit the description of the printer.
Station type

Select the station type from the drop-down list.

Choose from:

  • Windows printer: This setting finds the windows printer.
  • OPOS printer: This setting finds the OPOS printer, and requires you to select a remote host.
  • Hardware profile printer: This uses settings in the hardware profile
Font name This field will only become available if “Windows printer” is selected. Enter or edit the font name into the text box.
Edit Select this button to edit the font. This opens the Font dialog.
Windows printer Select the windows printer that you want to use from the drop-down list. Note that this is only available if “Windows printer” is selected in the “Station type” field.
Remote host Select a remote host from the drop-down list. Note that this is only available if “OPOS printer” is selected in the “Station type” field.
Edit Select this button to edit or add to the remote hosts. This opens the Remote hosts dialog.
OPOS device name Select the OPOS device name from the drop-down list.
View all restaurant stations Click on this link to go to Printing stations view.
