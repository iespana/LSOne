Bar code Dialog

Item > Items category > Items > Select retail item > Select Bar codes tab > Select Add button

What is the dialog used for?

  • Add a bar code to a retail item.
Fields & buttons Description
Bar code setup Select a bar code setup. To add a new one, close the dialog and select the Edit button next to the Bar code setup field. This opens the Bar code setup view.
Bar code Enter the bar code.
Variant

If the selected item is a retail item this field is disabled.

If the selected item is a header item you can select a variant if the bar code should sell a specific variant.

If the selected item is a variant item this field is pre populated.

Quantity Enter/Edit the quantity of items included in the bar code sale.
Unit Select the item unit in the bar code sale.
Add Select this button to add a new unit. This opens the Unit dialog.
Default bar code Select this check box to make this bar code the default for the item.
