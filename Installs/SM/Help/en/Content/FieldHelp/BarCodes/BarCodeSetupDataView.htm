Bar code Setup ID View

Setup > Bar codes > Barcodes > Bar codes setup > Highlight a line and select the Edit button

What is the view/dialog used for?

  • View/edit a bar code setup ID.
Fields & buttons Description
ID Displays the ID of the bar code setup.
Description View/edit the description of the bar code setup.
Bar code type

Select the symbology of the bar code mask setup:

  • No bar code
  • EAN128/UCC128
  • Code 39
  • Interleaved 2 of 5
  • Code 128
  • UPC A
  • UPC E
  • EAN13
  • EAN8
  • PDF417
  • Maxicode
Mask Select the bar code mask from the list.
Edit Select this button to edit the selected barcode mask in the bar code mask dialog.
Add Select this button to add a bar code mask. Opens the bar code mask dialog.
Maximum length Displays the maximum length of the bar code. (This depends on the bar code type).
Minimum length Displays the minimum length of the bar code. (This depends on the bar code type).
