Bar code Mask Setup View

Setup > Bar codes > Barcodes dropdown > Bar code mask setup

What is the view used for?

  • View and edit bar code mask setup lines.
  • Click and highlight a line in the upper pane to view the mask segments. Select the edit button to make changes and open the Bar code mask dialog.
List of Bar code masks

A list of all existing bar code masks

Fields & buttons Description
ID Displays the ID of the bar code mask setup.
Description Displays the description of the bar code mask setup.
Type

Displays the type of the bar code mask setup.

The options are:

  • None
  • Item
  • Customer
  • Employee
  • Coupon
  • Data entry
  • Sales person
  • Pharmacy
Symbology

Displays the symbology of the bar code mask setup.

The options are:

  • No bar code
  • EAN128/UCC128
  • Code 39
  • Interleaved 2 of 5
  • Code 128
  • UPC A
  • UPC E
  • EAN13
  • EAN8
  • PDF417
  • Maxicode
Prefix Displays the bar code mask prefix.
Mask Displays the bar code mask.
Length Displays how many characters are in the bar code mask, including the prefix.
Edit Highlight a line and select this button to edit the bar code mask setup. This opens the Bar code mask dialog.
Add Select this button to add a new bar code mask. This opens the Bar code mask dialog.
Delete Highlight a line and select this button to delete the bar code mask setup.
Segments in the selected mask

Displays a list of all the segments the selected bar code mask is made of.

Fields & buttons Description
Segment num The number / order of the segment in the bar code mask. You can change this by highlighting the mask segment lines and clicking on the green up and down buttons.
Type

Displays the type of bar code mask segment.

This includes:

  • Item
  • Any number
  • Check digit
  • Size Digit
  • Color digit
  • Style digit
  • EAN license code
  • Price
  • Quantity
  • Employee
  • Customer
  • Data entry
  • Sales person
  • Pharmacy
Length Displays the length of the bar code mask segment.
SegmentChar Displays the character used in the bar code mask to represent this segment.
Decimals Displays the number of decimals. This is only available for Price or Quantity types.
