Bar code Mask Dialog

Setup > Bar codes > Barcodes dropdown > Bar code mask setup > Select Add/Edit button

What is the dialog used for?

  • Create a new the bar code mask
  • Edit an existing bar code mask.
Fields & buttons Description
Description Enter/edit the description of the bar code mask.
Type

Select the type of bar code mask:

  • None
  • Item
  • Customer
  • Employee
  • Coupon
  • Data entry
  • Sales person
  • Pharmacy
  • Customized
  • Receipt barcode
  • Discount
  • Credit memo
  • Gift card
Symbology

Select the symbology for the bar code mask:

  • No bar code
  • EAN128/UCC128
  • Code 39
  • Interleaved 2 of 5
  • Code 128
  • UPC A
  • UPC E
  • EAN13
  • EAN8
  • PDF417
  • Maxicode
Mask

Here you can enter/edit the characters for the bar code mask but this field is automatically

Prefix Enter/edit the prefix characters.
Mask length Displays the mask character length.
Mask segments  
Segment num Displays the order in which the segment appears in the mask. 1 represents first, 2 second and so on. This can be changed by highlighting the line and clicking on the up or down arrow.
Type Displays the type the segment represents: Item Any number Check digit Size Digit Color digit Style digit EAN license code Price Quantity Employee Customer Data entry Sales person
Length Displays the length of the segment.
Character Displays the character used to represent the type of segment.
Decimals Displays the number of decimals in the segment. Note that this is only available for type Quantity and Price.
The segment can be moved forward in the bar code by highlighting the line and clicking on the up arrow.
The segment can be moved backwards in the bar code by highlighting the line and clicking on the down arrow.
Edit Highlight a line and select this button to edit the mask segment line in the Bar code mask segment dialog.
Add Select this button to add a mask segment. This opens the Bar code mask segment dialog.
Delete Highlight a line and select this button to delete the mask segment line.
