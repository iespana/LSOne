Suspended Transactions View

Sales > Transactions > Suspended drop down > Suspended transactions

What is the view used for?

  • View a suspended transaction.
Search filtering

Suspended transactions can be searched in a filtered search. One filtering line is added for each type of filter. You can even save your preferred filtering settings and they will be available when you open the Suspended transactions view.

Fields & buttons Description
  • Suspension type
  • Store
  • Terminal
  • Date

Select a filter type for each search filter line.
  • All
  • Default
  • Delivery
  • Modification
  • All: Search displays all suspended transactions no matter the suspension type.
  • Default: Search displays suspended transactions of suspension type Default.
  • Delivery: Search displays suspended transactions of suspension type Delivery.
  • Modification: Search displays suspended transactions of suspension type Modification.
Search field Enter the search criteria.

If there are more than one line press this button to delete a search criteria

You can also unselect the checkbox to exclude a criteria from the search

Press this button to add search criteria and narrow the filter.

The following actions are available from a drop down menu when this button is pressed

  • Save as default layout: The search lines you have selected will be saved to the view and opened when this view is opened
  • Load default layout: If you have edited the search settings since you last opened the view select this option to reload the saved layout
  • Reset: Select this option to reset the layout to be as displayed in image above
  • Toggle button locations: Select this option to set the and buttons in front of the filter type
Button:
Search		 Press this button or hit enter button on keyboard to search for an item with the filters set
Fields & buttons Description
Transaction date Displays the date of the suspended transactions.
Suspension type

Displays the suspension type. Suspension types are setup here: Suspended Types View
Transaction ID Displays the transaction ID.
Description Displays the description of the suspended transaction.
Store Displays the store that the transaction was suspended on.
Terminal Displays the terminal that the transaction was suspended on.
Delete Highlight a line and select this button to delete the suspended transaction.
Suspended transactions
Fields & buttons Description
Field Displays the suspended transaction field.
Data Displays the data from the suspended transaction.
