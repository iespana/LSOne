Suspension Field Dialog

Sale > Transactions >Suspended dropdown > Suspended types > Select a suspended type > Select Add button

What is the dialog used for?

  • This dialog is used for creating a new suspension field for an suspension transaction type.
Fields & buttons Description
Description Enter a description name for the new suspension type dialog. You can enter a maximum of 60 characters both numbers and letters
Type

Select the type for the suspension field:

  • Text
  • Customer
  • Name
  • Address
  • Infocode
  • Date
Infocode Selection

If the infocode is selected in the Type combo-box, the infocode selection combo-box is enabled.

An infocode can be chosen for the suspension field.
Required Check box for the required field.
