Add or Remove Customers

Customers > Groups > Groups > Add customers to the group (lower list)

What is the dialog used for?

  • This dialog is used add customers to the selected customer group.
  • In the dialog multiple customers can be added to the customer group at once.
  • Customers can also be removed from the customer group in this dialog.

Fields & buttons Description

Customers

In the dropdown you can search for customers by name or id.

You can use the button to select to search in the beginning of the name or ID or select contains to search anywhere in the name or ID.

In the upper list of the dropdown box you see a list of all customers in the group and the changes that have been made. You can delete customers from the group by pressing the

In the lower list the search results are displayed and you can select which customers to add to the group.

Press the close button when editing has been done.

Edit preview Displays a list of all customers in the group and any changes have been done do the group in the dialog.
Customer Displays the name of the customer that is in the group or has been added to the group.
Change Shows the change that has been done such as if the customer has just been added or if the customer has been removed from the group.
Button:
Images

The images in each line are buttons:

  • Press this button to delete this customer from the group
  • Press this button to revert the changes to this line. So if the customer had just been deleted it will not be deleted and if the customer had just been added to the group the line disappears from the list.
