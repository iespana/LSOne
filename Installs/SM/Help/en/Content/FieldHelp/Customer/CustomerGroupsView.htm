Customer Groups View

Customers > Groups > Groups

What is the view used for?

  • A list of all available customer groups and the customers in each group.
  • Customer groups allow to set a limit to how much discounts the customers in the group can get in a given period, day, week, month or year.

List of groups

Fields & buttons Description
Customer group

Displays the ID of the customer group.

Description

Displays the description of the customer group.

Category Displays the Category of the group. Categories can be created under Group categories or during the customer group creation. The category is just used for informational purposes for the customer groups.
Button:
Set as default		 Press this button to make the selected customer group the default customer group for all the customers in the group. If you only want to make the group the default group for a single customer you can do this from the Customer view, Customer group tab instead.
Edit

Press the edit button to edit the selected customer group in the Customer group view.

Add Press the add button to create a new Customer group. Opens the New Customer Group Dialog
Delete Press the delete button to delete the selected Customer group.

List of customers in the selected group

Fields & buttons Description
Customer

Displays the names of the customers in the group
Button:
View customer		 Opens the selected customer in the Customer view.
Edit

Press the edit button to edit the selected customer in the Customer view.

Add Press the add button to add customers to the customer group. Multiple customers can be added at once. Opens the Add or Remove Customers dialog
Delete Press the delete button to remove the selected customer from the group. Hold the Ctrl button while selecting to select multiple customers to remove from the group.
