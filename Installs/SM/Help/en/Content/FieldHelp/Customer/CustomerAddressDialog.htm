Customer Address Dialog

Customers > Customers > Edit a customer > Select Address tab > Select Add button or Edit button

What is the dialog used for?

  • Create a new address for the customer account or edit the selected customer address. Each customer can have three addresses with one of each address type. These are displayed on the POS when the customer is added to a transaction.

 

Fields & buttons Description
Address type

Select the type of address that you are adding to the customer:

  • Shipping
  • Billing
Address Enter the address.
City/State/Zip code Enter the City/State/Zip code if Generic with state is selected on the globe button.
Zip code/City Enter the Zip code/City if Generic without state is selected on the globe button.
Globe Select a different country's address format.
Country Select a country from the drop down menu.
Contact name Enter a contact name (could be different to the main customer name).
Phone Enter the contact's phone number.
Mobile phone Enter the contact's mobile (GSM) phone number.
Email address Enter the contact's email address.
Sales tax group Select a sales tax group for the customer.
