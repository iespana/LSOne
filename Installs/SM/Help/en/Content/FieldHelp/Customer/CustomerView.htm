Customer ID View

Retail > Customers > Customers > Highlight a customer and select Edit button

What is the view used for?

  • The Customer view is used to manage a customer in the Site Manager.

Note: The functionality to send out these emails is not included in the current version of LS One. Any such functionality would have to be done as a customization either by a 3rd party system or within the LS One suite. There are no plans to add this functionality to the next release of LS One.

Fields & buttons Description

Edit Click on this button to open Customer discount groups view.
Currency The currency that the discount is in.
Type The type of discount, line discount, for example.
Customer connection How the discount is connected to the customer, either it applies to all customers, a customer group or an individual customer.
Item connection The item that the discount is on. You can select an individual item, item group or all items.
Variant If the discount has been applied to a variant of the item, it will be shown here.
Unit The unit in which the item that has the discount is sold.
From date The date from which the discount becomes valid.
To date The date to which the discount is valid.
Amount The quantity of items that need to be sold in order to apply the discount.
Discount amount The discount amount.
Discount % (1) The primary discount percentage, if there are two discount percentages, this is the first to be taken off the item.
Discount % (2) The secondary discount percentage, this percentage is taken off the total amount after the first discount percent has been taken off. For example, if the items total value is 100, %1 is 10% and %2 is 10%, the calculation would be 100 – 10% = 90, then 90 – 10% = 81, so the end total of the item, after the discount would be 81.

View customer group Highlighting a line that has a Group in Customer Connection and then clicking on this button takes you to the price/discount groups view where you can view and edit the groups that the customer belongs to.
View item group Highlighting a line that has a Group in Item Connection and then clicking on this button takes you to the Item discount group view where you can view and edit the Item discount groups.

Loyalty tab
Fields & buttons Description
Issued The points issued on the card.
Used The points used on the card.
Expired The points expired on the card. This appears with a minus sign infront.
Points balance The current points status. The points that are currently available for use on the card.
Card number Displays the card number.
Type

Displays the type of tender issued on the card.

  • Card tender: Points can be used for payment, according to how it is set up in the scheme.
  • No tender: Points are not used for tender, but are collected and rewarded by the store.
  • Blocked: The card is blocked and cannot be used.
Scheme Displays the name of the loyalty scheme on this card. The loyalty scheme decides how long the points take to expire, the use limit of the points and the default multiplier type.
Current value  
Issued The points issued on the card.
Used The points used on the card.
Expired The points expired on the card. This appears with a minus sign in front.
Status The current points status. The points that are currently available for use on the card.
Edit Highlight a line and select this button to edit a loyalty card. This opens the Edit loyalty card dialog.
Add Select this button to select a loyalty card for the customer. This opens the Select loyalty card dialog.
Loyalty transactions tab
Fields & buttons Description
Filter
  • Date - Search for transaction in a time period.
  • Receipt number / Begins with - Search for transaction by receipt number. You can choose to search by Begins with or Contains.
  • Store - Find all loyalty transaction for the customer created in the selected store.
  • Terminal - Find all loyalty transaction for the customer created on the selected terminal.
  • Loyalty card - Find all loyalty transaction made with the selected card number. The card list only displays cards that belong to the open customer.
  • Scheme - Find all loyalty transactions created with the selected loyalty scheme.
  • Types - Filter the transaction down to the point status, expired, issued or used.
  • Status - filter the list to transaction that are opened or closed. The loyalty transactions close when all issued points are used.
