Manage reports view

Retail > Reports > Manage reports

What is the view used for?

  • In this list view you can import new reports, set where the reports are accessible or delete reports from the system.
Fields & buttons Description
Description This is the name of the report. This text gets imported with the report files.
Long description This is a description of the report used for more detailed information for the report buttons. This text gets imported with the report files.
Can be deleted

Some reports cannot be deleted from the system. These reports are system reports such as financial report that must exist in the system. You cannot delete such report but you can override them by uploading report with same ID for customization or localization purposes. System reports should not contain any report contexts

Other reports can be deleted and are marked in this field.

Edit Opens the Manage report dialog to edit in which context the selected report is active, that is where in the system the report is accessible.
Add Opens the Add report dialog to import new reports to the system.
Delete Deletes the selected report from the system.
