Trade Agreements - Sales Price Dialog

Retail > Price/discount > Price groups > Sales prices tab > Add/Edit button

What is the dialog used for?

  • Add a sales price to overwrite the base price of an item. This can be set specifically to customers, price groups or all.
Fields & buttons Description
Currency The sales currency that is valid for the sales price. For example if the company currency is in Euros and a sales price rule is set up for US Dollars, this rule will only work for stores which have their currency set to US Dollars.
Add Select this button to add a new currency. Opens the Currency dialog.
Item Select an item from the drop-down menu. *This field is not available in the item view.
Variant

This field is only available if a header item has been selected in the Item field. If the sales price should be valid for a single variant item select this variant item from the list.

If a header item has been selected in the Item field and no variant is selected in this field the offer will be available for all variant items.

Unit The unit of measure that the price is valid for. By default the item's sales unit is entered here but it is possible to limit the sales price to a specific unit.
Add Select this button to add a new unit. Opens the Unit dialog.
From date The date when the price should take effect.
To (date) The date when the price should stop being used (if nothing is selected the price is valid for an unlimited time period).
Quantity Enter the quantity that should be reached before the price becomes active.
Price Enter the sales price. If you enter a value into the "Price with tax" field and click in this field, the sales price according to the LS One Site Manager settings is calculated for you.
Price with tax Enter the price with tax. If you enter a value into the "Price" field and click in this field, the price with tax according to the LS One Site Manager settings is calculated for you.
Price misc. charges Enter a value for miscellaneous extra charges.
