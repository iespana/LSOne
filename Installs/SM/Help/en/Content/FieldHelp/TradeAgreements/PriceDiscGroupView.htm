Discount Groups View

Retail > Price/discount > Customer discount groups

What is the view used for?

  • to create and maintain customer discount group
  • add customers to the group
  • Create discounts for the group
Fields & buttons Description
Show

Select to view customers in and discounts for:

  • Line discount group
  • Multiline discount group
  • Total discount group
Group ID Displays the group ID.
Description Displays the group description.
Prices include sales tax

Displays:

  • Yes: Price does include sales tax
  • No: Price does not include sales tax

The default configuration for the LS One POS is that the price does not include tax. Each store can be configured so that the price in the LS One POS either includes tax or does not.

To configure the store for prices with or without tax:

  • Select operation Store setup > Stores > Stores;
  • Select the Store;
  • On the tab Settings, the field Price tax setting configures the setting for Prices with tax or Prices without tax.
Edit Highlight a line and select this button to edit the group description. Opens the Price group dialog.
Add Select this button to add a group. Opens the Price group dialog.
Delete Highlight a line and select this button to delete the group.
Customers tab

Displays customers in the group highlighted in the upper pane.

Fields & buttons Description
Account number Displays the customer account number.
Name Displays the customer's name.
Back Select this button to scroll back through customer list. Paging is only available if there are more than 500 customers listed.
Next Select this button to scroll to the next page of customers. Paging is only available if there are more than 500 customers listed.
Button: View Customer Highlight an item line and select this button to view it in the Customer ID view.
Add Select this button to add an item to the group. Opens the Customer search dialog.
Delete Click on a line and select this button to delete it.
Sales prices tab

View and edit the item and customer connections to the discount line.

Fields & buttons Description
Currency Displays the currency of the sales price.
Type

Shows the type of discount: Sales prices.
Item connection Displays the description of the Item or header item the sales price is valid for.
Customer connection This field always displays group as all the sales prices displayed in this list are valid for this group.
Variant This field is only visible if any of the sales prices include a variant item. Then this field displays the variant description of the item.
From date Displays date restrictions for the discount.
To date The date when the price should stop being used (if nothing is shown the price is valid for an unlimited time period).
Quantity The minimum quantity that is required for the item so the discount is activated. Zero value does not restrict the quantity.
Price The sales price of the item(s) in this price/discount group.
Price with tax The price with tax.
Button:
View item		 Click on a line and select this button to view the item connected. Opens Retail item ID view.
Edit Click on a line and select this button to edit it. Opens Trade agreements - Sales price dialog.
Add Create a new item line discount. Opens Trade agreements - Sales price dialog.
Delete Click on a line and select this button to delete it.
