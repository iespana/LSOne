Trade Agreements - Line Discount (Item) Dialog

Price/discount >Item discount groups > Discounts tab > Edit/Add button

What is the dialog used for?

  • Trade agreements Line discount is a discount for items given automatically in the POS when the item is sold as long as the line discount’s restrictions are met.
Fields & buttons Description
Currency Select the currency.
Add Select this button to add a new currency. Opens the Currency dialog.
Account code

Select an account code:

  • Customer: Discount is valid for a specific customer.
  • Group: Discount is valid for customers in a specific customer line discount group.
  • All: Discount is valid for all customers.
Account selection

Select a customer or customer group.

  • If Customer is selected, a list of all customers is displayed.
  • If Group is selected in Account code, a list of all customer line discount groups will be displayed.
  • If All customers is selected in Account code, this field is not valid.
From date Enter or select (from calender) date restrictions for the discount.
To (date) The date when the price should stop being used (if nothing is selected the price is valid for an unlimited time period).
Quantity The minimum quantity that is required for the item so the discount is activated. Zero value does not restrict the quantity.
Discount amount Enter a number if the discount should be a fixed amount.
Discount percentage 1 Enter a number for the discount percentage.
Discount percentage 2

This discount is combined with Discount percentage 1 When both Discount percentage 1 and 2 are used the system starts by discounting the amount with discount percentage 1 and then the new amount is discounted with discount percentage 2.

Example: discount percentage 1 is 10 and discount percentage 2 is 20 and the amount is 100. The new total after discount calculation will be 72

  • First the amount 100 is discounted with 10%, the amount is 90.
  • Then amount 90 is discounted 20% so the total amount is 72.
