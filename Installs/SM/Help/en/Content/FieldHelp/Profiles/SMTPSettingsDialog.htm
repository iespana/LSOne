E-mail Settings Dialog

Store setup > Profiles > Site service profiles > Edit a Site service profile > Click on configuration ... button > Click on SMTP setup button

What is the dialog/view used for?

  • To configure the e-mail sending settings for the Site Service.
Tab/Field name Description
Store  
Store This setting is not in use
SMTP server  
Server Enter the DNS name or IP address of the SMTP server
Port The port to use when communicating with the SMTP server
Authentication  
E-mail address The e-mail address that is used to log on to the SMTP server, and is therefore also the address that e-mail is sent from. In some cases, this may not be an e-mail address but rather a user name.
Password The password, if any, that is required to log on to the SMTP server
Display name The display name is optional, but may result in a nicer display on the receiver side. Instead of seeing only the e-mail address of the sender, most e-mail clients will show the display name.
E-mail settings  
Use SSL If checked, then communication with the SMTP server will be over the Secure Socket Layer protocol. Whether this is checked, depends solely on the setup of the SMTP server.
Text only If checked, then no attempts will be made to format outgoing e-mail. Note that it is less likely that e-mail will be flagged as spam if this setting is not checked!
Signature Any signature footer to add to outgoing e-mail can be entered here
Button:
Send test e-mail		 Click on this button to test sending an e-mail to an e-mail address you enter in the Send test e-mail message dialog that opens.
