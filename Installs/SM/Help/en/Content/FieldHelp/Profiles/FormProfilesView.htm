Form Profiles View

Store setup > Profiles > Form profiles

What is the dialog/view used for?

  • View a list of the form profiles, add, edit or delete a form profile on the Site Manager, add, edit or delete form layouts.

Form profile consist of a set of different types of forms. Each form is printed in different operation in the LS POS.

Note that the default form profile and its form layouts can not be deleted.

List of form profiles

A list of all existing form profiles.

Fields & buttons Description
Form profile Contains the name of the form profile.
System profile Specifies if the profile is the default system profile.
Edit Opens the Form profile dialog where you can edit the description of the form profile.
Add Opens the Form profile dialog to add a new form profile.
Delete Deletes the selected form profile from the list.
Form layouts in form profile

List of all the form layouts the selected form profile has.

Fields & buttons Description
Form type Displays the type of the form. The type defines when the selected layout should be printed.
Layout Displays the description of this form type.
Print as slip Displays if the form will be printed on the slip printer or normal receipt printer. Not all receipt printers have a slip printer.
Print behavior

Displays the print behavior set on the layout.

  • Always print - A form with this selection always gets printed.
  • Not print - A form with this selection never gets printed.
  • Ask customer - If a form has ask customer selected the LS One POS asks the POS user first.
System layout Specifies if the layout from the profile was created by the system.
Edit Opens the Form profile line dialog showing the Type and Layout of the line selected.
Add Opens the Form profile line dialog with empty Type and Layout fields.
Delete Deletes the selected form profile line.

Actions

  • Form layouts - Select this link to open the form layouts view to edit, add or delete a form layout.
  • Reset default profile - Resets all changes made to the default system profile and form layouts created by the system.
