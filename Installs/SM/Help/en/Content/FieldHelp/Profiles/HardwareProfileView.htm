Hardware Profile ID View

Store Setup > Profiles > Hardware profiles > Edit a hardware profile

What is the view used for?

  • To add and edit hardware profiles.
  • Hardware profiles are set on terminals to connect to hardware such as Line display and Printers. If all terminals have similar hardware you only need to create one hardware profile.
Fields & buttons Description
ID

Displays the ID of the hardware profile.
Description Displays the description of the hardware profile.
Drawer

Connection information for a cash drawer.

Fields & buttons Description
Drawer connected Select this to activate the drawer connection for the POS. When you mark the field all other fields are activated.
Device name Select the drawer device name from the list view. Or if the POS is not running on the same machine as the Site Manager type in the correct OPOS device name.
Description Enter a description for the device.
Open text Enter a text that will be displayed on the customer line display while the drawer is open.
Line display tab

information about the tab if needed

Fields & buttons Description
Line display connected

Select from the list which type of line display to connect to:

  • None - No Line display is connected
  • OPOS - an OPOS hardware device has to be connected to the POS.
  • Virtual display - a dialog will pop up on the screen and display all line display messages.
Device name If an OPOS device has been connected select the hardware setup. Or if the POS is not running on the same machine as the Site Manager type in the correct OPOS device name.
Description Enter a description for the line display setup.
Display total text Enter text to display on the line display next to the total amount. For example, “Total.”
Display balance text Enter text to display on the line display next to the balance amount. For example, “Balance.”
Till closed – line 1 Enter text to display on the first line of the line display when the till is closed. For example, “Till is closed.”
Till closed – line 2 Enter text to display on the second line of the line display when the till is closed. For example, “Sorry for the inconvenience.”
Line display charset Type in the charset number that your device supports.

Fields & buttons Description
Dual display in use Select this check box if you want to have a dual display.
Description Type in a description for the device.
Receipt width Enter a percentage for the receipt width. The rest of the screen will
Screen number Select which screen the dual display should open at. Check which screen the LS One POS opens at in the visual profile.

Card reader tab

Hardware needed: Card reader (MSR).

If you have an OPOS device you need to activate the functionality, select the device name and fill in Start, Differentiating and Final digit.

If you have a keyboard MSR reader you only have to fill in Start, Differentiating and Final digit.

Fields & buttons Description
Card reader connected Select this check box to connect the card reader to the terminal.
Device name Select the device name from the list. Or if the POS is not running on the same machine as the Site Manager type in the correct OPOS device name.
Description Enter a description for the device.
Start digit Enter the start digit your card reader uses. To check which character this is you can open Notepad and swipe any card. The first character entered in notepad is your start digit.
Differentiating digit Enter the differentiating digit your card reader uses. The differentiating digit is to divide between track 1 and track 2 of a card number. To check which character this is you can open Notepad and swipe any credit card. The character entered in between you card number and the expiration date in notepad is your differentiating digit.
Final digit Enter the final digit your card reader uses. To check which character this is you can open Notepad and swipe any card. The last character entered in notepad is your finish digit.
Printer tab

Hardware: OPOS receipt printer or a Windows printer.

If you have not printer you can use the Virtual printer for testing purposes.

Fields & buttons Description
Printer connected Select the printer connected from the list menu: None – no printer is connected to the POS OPOS – And OPOS printer has to be connected to the POS to work. Windows printer – The POS sends receipts and other printouts to a windows printer Virtual printer - the LS One POS opens a dialog where the receipt will print.
Device name Select the device name from the list menu. Or if the POS is not running on the same machine as the Site Manager type in the correct OPOS device name.
Description Enter a description for the device.
Use decimal conversion Select this check box to use decimal conversion.
Printer charset Type in the charset number that your device supports.
Extra lines before cut Enter the number of empty line feeds to print before cutting the receipt. Sometimes the printer almost cuts off the last row of text of the receipt then you can add empty line feeds here to make more bottom margin.
Windows printing configurations
Left margin (mm) Enter the size of the left margin in mm. Use this to control where the information is printed on the paper.
Right margin (mm) Enter the size of the right margin in mm. Use this to control where the information is printed on the paper.
Top margin (mm) Enter the size of the top margin in mm. Use this to control where the information is printed on the paper.
Bottom margin (mm) Enter the size of the bottom margin in mm. Use this to control how long receipt can be printed on one piece of paper. If the receipt is too long it will be printed on multiple pages.
Font The font is fixed to Courier new as that is monospace font and the text strings positions are calculated in number of characters.
Font size Enter the size of the font in pixels.
Print design boxes Selecting this option will show the border of all fields included in the form being printed on a windows printer.
Folder location Press the ... button to browse to a folder to save email receipts in.
