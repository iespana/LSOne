User Profiles View

Users > Users> User profiles

What is the view used for?

  • View a list of the user profiles on the Site Manager, add and edit user profiles.
User profiles
Tab/Field name Description
Description The description of the user profile.
Store The store of the user profile.
Visual profile The visual profile of the user profile.
Layout The touch button layout of the user profile.
Language code The language code of the user profile.
Keyboard language The keyboard language of the user profile.
Discount permissions Displays a check if the user profile has any discount settings different than 0.
Edit Highlight a line and select this button to edit a user profile. Opens User profile view.
Add Select this button to add a new user profile. Opens New user profile dialog.
Delete Highlight a line and select this button to delete a user profile.
[Link] Edit store Edit the store of the selected user profiles. Opens Edit user profile dialog.
[Link] Edit visual profile Edit the visual profile of the selected user profiles. Opens Edit user profile dialog.
[Link] Edit layout Edit the touch button layout of the selected user profiles. Opens Edit user profile dialog.
