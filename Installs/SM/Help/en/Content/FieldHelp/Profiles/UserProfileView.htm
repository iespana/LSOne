User Profile View

Users > Users> User profile

What is the view used for?

  • Edit the user profile settings.
Discount settings tab

View and edit the discount settings for the user profile.

Fields & buttons Description
Maximum line discount amount The maximum amount that the user is allowed to discount on a line in a transaction on the POS.
Maximum line discount percentage The maximum percentage that the user is allowed to discount on a line in a transaction on the POS.
Maximum total discount amount The maximum amount that the user is allowed to discount on a transaction on the POS.
Maximum total discount percentage The maximum percentage that the user is allowed to discount on a transaction on the POS.
Maximum line return amount The maximum amount that the POS user is allowed to return on a line.
Maximum total return amount The maximum amount that the POS user is allowed to return on a transaction.
Other tab

View and edit other settings for the user profile.

Fields & buttons Description
Store Choose the store that you want to assign the POS user to from the the drop-down menu.
Edit Edit the selected store.
Visual profile Choose the visual profile that you want to assign to the POS user from the drop-down menu. If you leave this blank it will automatically assign the storeâ€˜s visual profile.
Edit Edit the selected visual profile.
Touch button layout Choose the touch button layout that you want to assign to the POS user from the drop-down menu. If you leave this blank it will automatically assign the storeâ€˜s touch button layout.
Edit Edit the selected touch button layout.
Language code Select a language from the drop down list for the POS user. If you leave it on Default, it will be set to the storeâ€˜s language.
Keyboard language Choose the keyboard language from the drop-down list.
Users tab

View and edit the users assigned to the user profile.

Fields & buttons Description
Username The username.
Login The login of the user.
User groups The user groups of the user.
Name on receipt The name on receipt of the user.
Enabled True if the user is enabled.
Add Add users to the user profile
Delete Delete users from the user profile.
