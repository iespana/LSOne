Mix and Match Line Dialog

Site Manager > Retail > Price/discount > Discount offers > Mix and match > Highlight a disabled mix and match offer > Select Add/Edit button in Mix and match lines section

What is the dialog used for?

  • To add items, variants or retail groups to a mix and match offer.
  • To edit an existing offer line.
Fields & buttons Description
Type

Select one of the following to determine the type that the offer will be valid for:

  • Item
  • Retail group
  • Retail department
  • Special group
Item/Retail group/Retail department
/Special group		 The name and content of this field depends on the option selected in the field Type. Select which item / retail group / retail department / special group the mix and match offer is valid for from the selection box.
Variant

This field is only available if the Type field is set to item and a header item has been selected in the Item field. If the offer should be valid for a single variant item select this variant item from the list.

If a header item has been selected in the Item field and no variant is selected in this field the offer will be available for all variant items.

Line group Choose a line group from the list. The line group defines how many items in this group have to be purchased to trigger the mix and match offer.
Price / discount  
Standard price Displays the standard price. That is the base sales price set on the item.
Standard price with tax Displays the standard price including tax.
Discount type Displays the type of the offer.
Deal price Displays the deal price if this mix and match offer is of that type.
