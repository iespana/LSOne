Discount Offers View

Site Manager > Retail > Price/discount > Discount offers

What is the view used for?

  • View and edit the discount offers.
  • Activate and deactivate discount offers.
  • Add discount offer lines to discount offers.
Discount offers - upper pane

In the upper list all discount offers are listed.

Fields & buttons Description
Offer number Displays the ID of the discount offer.
Description Displays the discount offer’s description.
Status Displays the discount offer’s status: Enabled means that the offer is active and in use, Disabled means that the offer is inactive and available for editing. (See Button: Disable/Enable for more information).
Discount % Displays the discount offer percentage.
Priority

Displays the priority of the offer.

The priority of the offer controls which offer is triggered if the item exists in more than one offer of the same type. The lower the priority number, the higher the priority of the offer. For example, if the priority number is 1, it will have a higher priority than an offer that has the priority number 2. An overview of all discount priorities can be seen in periodic discount view.
Validation period Displays the Validation period. Validation periods are set up in the validation periods view.
Starting date Displays the starting date of the validation period.
Ending date Displays the ending date of the validation period.
Price group Displays the price group set to the discount offer. The discount offer is active in the stores in the selected price group. If the field is empty the offer is valid in all stores.
Customer connection Displays a customer group or a single customer the offer is valid for. If this field is empty the offer is valid for all customers.
Triggering

Displays if the offer is:

  • Automatic: that is the discount is given automatically as soon as all the items needed to trigger this discount have been sold.
  • Manual: to trigger the discount the user needs to press a button on the POS. The operation of the button should be set to Manually trigger periodic discount. When this button is pressed all periodic discounts that are set to manual and apply to this transaction are activated.
Disable/Enable Highlight a line and press this button to deactivate or activate a discount offer. If the line displays the offer’s status as Enabled, the button will be set to deactivate. If the offer’s status is Disabled the button will be set to activate. Note that you can only edit an offer if the line is disabled and it will only work on the POS if the line is enabled.
Edit

Highlight a line and press this button to edit the configuration of the discount offer in the discount offer dialog. This includes setting its priority, discount, price group and validation period.

Add Select this button to create a new discount offer.
Delete Highlight a line and press this button to delete the selected discount offer.
Discount offer lines - lower pane

In the lower list the lines of a selected discount offer are displayed.

Fields & buttons Description
Type Displays the type of discount offer line. This can be: Item (retail item, header item or variant item) Retail group, Retail department, All, or Special group.
Item relation This is the ID number of the type, if it is an item, it will be the item ID, if it is a Retail group, the retail group’s ID is displayed, and so on.
Description Displays the description of the line type (Item (retail item, header item or variant item), Retail group, Retail department, All, or Special group).
Variant This field is only visible if any of the lines include a variant item. Then this field displays the variant description of the item.
Standard price with tax Displays the standard price with tax included.
Discount % Displays the discount percentage on the line.
Offer price with tax Displays the price including tax and discount.
Discount amount with tax Displays the discount amount with tax included.

Button:

Edit all lines

 Opens the Edit all lines dialog where multiple discount offer lines can be added or deleted at the same time.
Edit Highlight a discount offer line and select this button to edit the line.
Add Select this button to create a new discount offer line.
Delete Highlight a discount offer line and select this button to delete a line.
