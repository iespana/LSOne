Validation Period Dialog

Site Manager > Retail > Price/discount > Validation periods > Select Add/Edit button

What is the dialog used for?

  • All periodic discounts and promotions can have validation periods. The validation periods control when the periodic discounts and promotions are valid.
  • The validation periods can be set up to be valid for:
    • Specific hours of the day (happy hour);
    • Specific days of the week (weekend offer);
    • A specific time period (a month, spring, Easter);
    • A combination of these factors.
  • Note: If no validation period is assigned to periodic offers, the offers are always valid.

If no validation period is assigned to periodic offers, the offers are always valid.

Fields & buttons Description
Validation period  
Description Enter or edit the description of the validation period.
Default settings Set the validation period with default settings and then you can tailor each day in Weekday period settings. If, for example, you want to create a weekday happy hour, you can set the default settings and change Saturday and Sunday in the weekend settings to Never.
Starting date Select the starting date of the validation period by clicking on the calendar icon and selecting a date or typing the date in manually. Note that the default data has date settings: DD/MM/YYYY.
Ending date Select the ending date of the validation period by clicking on the calendar icon and selecting a date or typing the date in manually. Note that the default data has date settings: DD/MM/YYYY.
Time duration
  • Always: Select this option if you want the validation period to be valid at all hours of the day between the dates set.
  • Period: Select this option if you want the validation period to be restricted to a certain time during the day.
Starting time Select a starting time for the validation period’s time. This field is only enabled if Period has been selected in the Time duration field.
Ending time Ending time Select an ending time for the validation period’s time.This field is only enabled if Period has been selected in the Time duration field.
Time within bounds

If you select this check box, then between the starting and ending time, the discount offer is valid. If it is clear, then between the starting and ending time, the discount offer is not valid but then it is valid other times of the day.
Ending time after midnight Select this check box if the offer ends after midnight.
Weekday period settings

If you want to make the validation period specific to days of the week, use this box to configure those settings.

To set up multiple days, such as a Monday to Thursday Happy Hour, you have to select each day from the dropdown menu, and select Time Duration: Always or Period, depending on whether you want the offer to be valid for a long time or a selected period. You must do this for each day that you want to select.

Day Select the day that you want to edit. In the section names: Weekdays currently using special settings, the days of the week are marked with a tick if they have any settings that are not disabled.
Time duration
  • Disabled: Select this option if you want the day to follow the validation period in the default settings.
  • Always: Select this option if you want the day to have this discount all day, despite any period settings in the default settings.
  • Never: Select this option if you want the day not to have the discount period in the default settings.
  • Period: Select this option to set a period for the day that is different to the default settings.
Starting time Select a starting time for the validation period’s time for the selected day. This field is only enabled if period has been selected in the Time duration for the selected day.
Ending time Select an ending time for the validation period’s time for the selected day. This field is only enabled if period has been selected in the Time duration for the selected day.
Time within bounds If you select this check box, then between the starting and ending time, the discount offer is valid. If it is clear, then between the starting and ending time, the discount offer is not valid. This field is only enabled if period has been selected in the Time duration for the selected day.
Ending time after midnight Select this check box if the offer ends after midnight on the selected day. This field is only enabled if period has been selected in the Time duration for the selected day.
Weekdays currently using special settings  
Mon / Tue / Wed / Thu / Fri / Sat / Sun If the check box is marked with a tick, then that day has special weekday period settings that are different to the default settings.
