Promotion Dialog

Site Manager > Retail > Price/discount > Promotions > Select Add button

What is the dialog used for?

  • Create or edit a promotion.
  • Configure the default discount for the promotion price. The discount is calculated from the itemsâ€™ retail price.
Fields & buttons Description
General  
Description Enter or edit the description of the promotion.
Default discount Enter or edit the default discount percentage. Whether the discount is from the price or the price including tax depends on the store setup.
Price group Select a price group from the options available on the list. The promotion will only be active in the stores in the selected price group. Edit the price groups in: Site Manager > Retail > Price/discount > Price/discount groups. If no price group is selected the promotion will be active in all stores.
Add Select this button to add a new price group to the list.
Account code Select an account code:
  • None - The promotion is valid for all customers.
  • Customer - The promotion is valid for the customer selected in next field.
  • Group - The promotion is valid for all customers in the customer group selected in next field.
Account selection Select a customer or customer group the promotion should be valid for (depending on the Account code).
Triggering

Displays if the offer is:

  • Automatic: that is the discount is given automatically as soon as all the items needed to trigger this discount have been sold.
  • Manual: to trigger the discount the user needs to press a button on the POS. The operation of the button should be set to Manually trigger periodic discount. When this button is pressed all periodic discounts that are set to manual and apply to this transaction are activated.
Validation period  
Validation period Select the validation period from the list.
Edit Select to edit the selected validation period in the validation period dialog.
Add Select the add button to add a new validation period. Open the validation period dialog.
Starting date Displays the starting dates for the validation period. To edit, select the Validation period edit button.
Ending date Displays the end dates for the validation period. To edit, select the Validation period edit button.
