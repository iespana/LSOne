Edit All Lines Dialog

Retail > Price/discount > Discount offers > Edit all lines button

What is the dialog used for?

  • To add and delete multiple lines for a discount offer at once. All the lines added to an offer this way get the default discount set in the discount offer header line.
Fields & buttons Description
Type

Select which type of lines to add to the discount offer

  • Item
  • Retail group
  • Retail department
  • Special group
Relation

Select one or many retail items/header items/variant items/retail groups/retail departments/special groups (depending on the selection in the type box) to add to the discount offer.

Edit preview  
Type Displays the type of the discount line.
Description Displays the name of the item, retail group, retail department or special group that is included in the discount offer.
Variant This field is only visible if any of the lines includes a variant item. Then this field displays the variant description of the item.
Change

Displays what action has been taken since the dialog was opened.

  • None - The line was included in the offer before.
  • Add - you have just added the line.
  • Delete - The line was included in the offer before but you have deleted it.
Action icon

Select this button to revert the changes you have made for the line.

Select this button to delete the line from the discount offer.
