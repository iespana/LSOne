Mix and Match View

Site Manager > Retail > Price/discount > Mix and match

What is the view used for?

  • View and edit mix and match offers.
  • Activate and deactivate mix and match offers.
Mix and match offers - upper pane

In the upper list all mix and match offers are listed.

Fields & buttons Description
Offer number Displays the ID of the mix and match offer.
Description Displays the mix and match offer’s description.
Status Displays the mix and match offer’s status: Enabled means that the offer is active and in use, Disabled means that the offer is inactive and available for editing. (See Button: Disable/Enable for more information).
Priority

Displays the priority of the offer.

The priority of the offer controls which offer is triggered if the item exists in more than one offer of the same type. The lower the priority number, the higher the priority of the offer. For example, if the priority number is 1, it will have a higher priority than an offer that has the priority number 2. An overview of all discount priorities can be seen in periodic discount view.
Discount type Displays the discount type:
  • Deal price:
    The items required in the mix and match offer are sold for a fixed price that is defined in the Deal price field.
  • Discount %:
    When an item that triggers the mix and match offer is selected, this percentage is discounted from its total value.
  • Discount amount:
    When the mix and match offer is triggered, the value entered here is discounted from the item’s total price.
  • Least expensive:
    The discount, which equals the price of the least expensive item, is then proportionally distributed to all of the lines.
  • Line specific:
    The price and discount are set up with the offer lines. The mix and match lines can be set up individually and they do not need to be identical.
Value Displays the discount value. Note that no number is displayed if the discount is line specific.
Number of lines to trigger Displays the number of items to be sold to trigger for the mix and match offer on the POS.
Validation period Displays the Validation period. Validation periods are set up in the validation period dialog.
Price group Displays the price group set to the mix and match offer. The mix and match offer is active in the stores in the selected price group. If the field is empty the offer is valid in all stores.
Customer connection Displays a customer group or a single customer the offer is valid for. If this field is empty the offer is valid for all customers.
Triggering

Displays if the offer is:

  • Automatic: that is the discount is given automatically as soon as all the items needed to trigger this discount have been sold.
  • Manual: to trigger the discount the user needs to press a button on the POS. The operation of the button should be set to Manually trigger periodic discount. When this button is pressed all periodic discounts that are set to manual and apply to this transaction are activated.
Button:
Disable/Enable

Highlight a line and press this button to deactivate or activate a mix and match offer. If the line displays the offer’s status as Enabled, the button will be set to deactivate. If the offer’s status is Disabled the button will be set to activate.

Note that you can only edit an offer if the line is disabled and it will only work on the POS if the line is enabled.
Button:
Line groups

Select the Line groups button to create line groups for the Mix and match offer.

For each Mix and match offer you need to create new line group(s). The line group decides how many units of the items assigned to it need to be sold to trigger the offer.

You can put many items in the same line group or just one. You can also set up many line groups in the same Mix and match offer, allowing you to combine quantities in the offer.

Edit Highlight a line and press this button to edit the configuration of the mix and match offer in the Mix and match dialog. This includes setting its priority, discount, price group and validation period.
Add Select this button to create a new mix and match offer.
Delete Highlight a line and press this button to delete the selected mix and match offer.
Mix and match lines - lower pane

In the lower list the lines of a selected mix and match offer are displayed.

Fields & buttons Description
Line group Displays the color of the line's line group
Type Each line is of a specific type. The options for Type are Item, Variant and Retail group.
Item relation This is the ID number of the type, if it is an item, it will be the item ID, if it is a Retail group, the retail group’s ID is displayed, and so on.
Description Displays the description of the line type (Item , Retail group, Retail department, or Special group).
Variant This field is only visible if any of the lines include variant items. Then this field displays the variant description of the item.
Line group Displays the description of the line group.
Price unit The price of a unit, note that this is only displayed if the type is a variant or an item.
Price unit including tax The price of a unit including tax, note that this is only displayed if the type is a variant or an item.

Button:

Edit all lines

 Opens the edit all lines dialog where multiple mix and match offer lines can be added or deleted at the same time.
Edit Highlight a line and select this button to edit the line by opening the Mix and match line dialog.
Add Select this button to create a new line by opening the Mix and match line dialog.
Delete Highlight a line and select this button to delete the selected line.
