Discount Offer Line Dialog

Site Manager > Retail > Price/discount > Discount offers > Highlight line > Select Add/Edit button in Discount offer lines area

What is the dialog used for?

  • To add items, header items, variant items, retail groups, retail departments or special groups to a discount offer.
  • To edit an existing offer line
Fields & buttons Description
Type

Select one of the following to determine the type that the offer will be valid for:

  • Item (Retail item, header item or variant item)
  • Retail group
  • Retail department
  • All
  • Special group
Item/Retail group/Retail department
/Special group		 The name and content of this field depends on the option selected in the field Type. Select which item / retail group / retail department / special group the discount offers is valid for from the selection box.
Variant

This field is only available if the Type field is set to item and a header item has been selected in the Item field. If the offer should be valid for a single variant item select this variant item from the list.

If a header item has been selected in the Item field and no variant is selected in this field the offer will be available for all variant items.

Price / discount  
Standard price Displays the standard price. That is the base sales price set on the item.
Standard price with tax Displays the standard price including tax.
Discount percent View or edit the discount percent here. (If you edit this field the others will change to reflect this).
Discount amount View or edit the discount amount here. (If you edit this field the others will change to reflect this).
Discount amount with tax View or edit the discount amount with tax here. (If you edit this field the others will change to reflect this).
Offer price View or edit the offer price here. (If you edit this field the others will change to reflect this).
Offer price with tax View or edit the offer price with tax here. (If you edit this field the others will change to reflect this).
