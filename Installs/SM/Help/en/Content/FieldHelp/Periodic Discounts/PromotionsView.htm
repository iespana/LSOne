Promotions View

Site Manager > Retail > Price/discount > Promotions

What is the view used for?

  • Promotions are set up so that the LS POS is able to sell items on a promotional price in a given period and with no manual intervention. A promotion is a price change which can be set up as a discount of the retail price or a fixed price for the item.
  • The system can have an unlimited number of promotional offers where each promotional offer is set up separately. Each promotional offer has a validation period and set of offer lines which can be of several types, for example Item and Retail groups.
  • To edit a promotion it has to be deactivated/disabled.
Discount offers - upper pane

In the upper list all discount offers are listed.

Fields & buttons Description
Offer number Displays the ID of the promotion.
Description Displays the description of the promotion.
Status Displays the promotion's status: Enabled means that the offer is active and in use, Disabled means that the offer is inactive and available for editing. (See button: Disable/Enable for more information).
Validation period Displays the Validation period. Validation periods are set up in validation periods dialog.
Starting date Displays the starting date of the validation period.
Ending date Displays the ending date of the validation period.
Price group Displays the price group set to the promotion. The promotion is active in the stores in the selected price group. If the field is empty the offer is valid in all stores.
Customer connection Displays a customer group or a single customer the promotion is valid for. If this field is empty the promotion is valid for all customers.
Button:
Disable/Enable 		Highlight a line and press this button to deactivate or activate a promotion. If the line displays the offer’s status as Enabled, the button will be set to deactivate. If the promotions’ status is Disabled the button will be set to activate. Note that you can only edit an promotion if the line is disabled and it will only work on the POS if the line is enabled.
Edit Highlight a line and press this button to edit the configuration of the promotion in the promotion dialog. This includes setting its discount, price group and validation period.
Add Select this button to create a new promotion.
Delete Highlight a line and press this button to delete the selected promotion.
Discount offer lines - lower pane

In the lower list the lines of a selected discount offer are displayed.

Fields & buttons Description
Type Displays the type of promotion line. This can be: Item (retail item, header item or variant item) Retail group, Retail department, All, or Special group.
Item relation This is the ID number of the type, if it is an item, it will be the item ID, if it is a Retail group, the retail group’s ID is displayed, and so on.
Description Displays the description of the line type (Item (retail item, header item or variant item), Retail group, Retail department, All, or Special group).
Variant This field is only visible if any of the lines include a variant item. Then this field displays the variant description of the item.
Standard price with tax Displays the standard price with tax included.
Discount % Displays the discount percentage on the line.
Promotion price The promotion price of an item which is only displayed when line is of type Item or Variant.
Promotion price with tax Displays the promotion price including tax and discount.
Discount amount Displays the discount amount.
Discount amount with tax Displays the discount amount with tax included.
Edit Highlight a promotion line and select this button to edit the line in the promotion line dialog.
Add Select this button to create a new promotion line in the promotion line dialog.
Delete Highlight a promotion line and select this button to delete a line.
