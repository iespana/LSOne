Combinations of Item Variations Dialog

Retail > Retail item > Select item > Variation combinations tab > Button: Create combinations

What is the dialog used for?

  • When you have selected the groups that you would like to use from the drop-down menus in the variations combinations tab view, you can click on the button, “Create combinations” and the Site Manager will automatically create all of the possible combinations out of these groups. You can then select any number of these combinations from the list.
  • Note: the item must have variations set to it for the variation groups to be available on the retail item card.

This dialog does not exist in LS One version 2016 or later.

Fields & buttons Description
Selection This column shows you which combinations have been selected.
Size This column shows you which sizes are available.
Color This column shows you which colors are available.
Style This column shows you which styles are available.
Select all Click on this button to select all variation combinations on the list.
Deselect all Click on this button to deselect all variation combinations on the list.
