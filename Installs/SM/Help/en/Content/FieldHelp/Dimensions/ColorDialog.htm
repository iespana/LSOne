Color Dialog

Site Manager > General Setup > Variations > Colors

What is the dialog used for?

  • Edit an existing color or create a new one.
  • Note: The variation does not have to be a Size, Color or Style, these are just the descriptions available, however you can create flavors, for example, and put it in one of the available categories.

This dialog does not exist in LS One version 2016 or later.

Fields & buttons Description
ID This is an automatic ID number or code that the Site Manager generates.
Name This is the name of the color that appears on the Site Manager and the POS.
Add This button allows you to add a new color.
Delete Highlight a color by clicking on it once and press the delete button to erase it from the existing colors list.
Name Highlight a color on the list or click on the add button and you will be able to edit an existing color name or create a new color.
Description Highlight a color on the list or click on the add button and you will be able to edit an existing color description or create a new one. Enter additional information into this box. Note that you will only be able to see it in this context, it does not appear on the POS.
Save Click on the Save button to save the color.

Cancel

 Click on the Cancel button to cancel the changes you have made.
