Style Groups View

Site Manager > General Setup > Variations > Style Groups

What is the view used for?

  • In the Style groups view you can add style groups and add styles to the style groups.

This view does not exist in LS One version 2016 or later.

List of style groups

A list of all existing style groups.

Fields & buttons Description
Style group The code assigned to the style group by the Site Manager.
Description The name of the style group.
Edit Click here to edit the description of the style group in the style group dialog. Select a style group to make this button available.
Add Click here to add a new style group in the style group dialog. Click on a style group to make this button available.
Delete Click here to delete a style group. Select a style group to make this button available.
Styles in group

A list of all styles in the selected in the list of all style groups.

Fields & buttons Description
Style Select a style from the drop-down-list
Name The name of the style
Number in barcode Displays an ID number for this style that is used to sell an item with this style.
Edit Click here to edit a style. Click on a style to make this button available.
Add Click here to add a new style in the style group line dialog.
Delete Click here to delete a style. Click on a style to make this button available.
