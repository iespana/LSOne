Variation Combination Dialog

Retail > Retail item > Select item > Variation combinations tab > Select Add button

What is the dialog used for?

  • Press the add button at the bottom-right of the screen in the Retail item card’s tab “Variation combinations” and the Variation Combination window pops up. Create individual variation combinations based on the variation groups that are available to this item.

This dialog does not exist in LS One version 2016 or later.

Fields & buttons Description
Name Enter a description of the variation combination.
Size Select a size from the drop-down list menu.
Edit Click on edit to edit existing or create new size.
Color Select a color from the drop-down list menu.
Edit Click on edit to edit existing or create new color.
Style Select a style from the drop-down list menu.
Edit Click on edit to edit existing or create new style.
