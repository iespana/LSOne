Parked Inventory

General setup > Inventory > Parked

What is the view used for?

  • Manage parked inventory.

All fields and search filters are the same as for inventory adjustments except for the following:

  • Status search field allows you to search for reservations that are either Active, Posted or All (which is the default setting)
  • Archive document option is not available; instead there is a Move to inventory option that allows you to move all items in the selected parked inventory documents to be moved to main inventory.
Search filter

Parked inventory journals can be searched for in a filtered search. One filtering line is added for each type of filter. You can even save your preferred filtering settings and they will be available when you open the parked inventory view.

 

Fields & buttons Description
  • Status
  • Created date
  • Description
  • Store

Select a filter type for each search filter line.

Check the box to use the search criteria.
  • Status

Selecting this options allows you to search for reservations that are either Active, Posted or All (which is the default setting)

  • Active: Selecting this option allows you to search for parked inventory journals that are active.
  • Posted: Selecting this option allows you to search for parked inventory journals that have been archived.
  • All: Selecting this option will list all parked inventory journals in the system.
  • Description
  • Begins with: Selecting this option allows you to search for the beginning of a value in the fields Description. (This is the default setting)
  • Contains: Selecting this option allows you to search for the number or letters in fields Description. This field is only visible for types that allow free text search.
Search field Enter the search criteria. For description you can enter a free text.

If there are more than one line press this button to delete a search criteria.

You can also deselect the checkbox to exclude a criteria from the search.

Press this button to add search criteria and narrow the filter.

The following actions are available from a drop down menu when this button is pressed.

  • Save as default layout: The search lines you have selected will be saved to the view and opened when this view is opened.
  • Load default layout: If you have edited the search settings since you last opened the view select this option to reload the saved layout.
  • Reset: Select this option to reset the layout to be as displayed in image above.
  • Toggle button locations: Select this option to set the and buttons in front of the filter type.
Button:
Search		 Press this button or hit enter button on keyboard to activate a search with the filters set.
