Generate from an existing stock counting journal

Inventory > Stock > Counting > Generate from an existing stock counting journal

What is the dialog used for?

  • Create a new stock counting journal generated from a previous one.

Stock counting journals can be searched for in a filtered search.

Search filter

Stock counting journals can be searched for in a filtered search. One filtering line is added for each type of filter. You can even save your preferred filtering settings and they will be available next time you open the view.

 
Fields & buttons Description
Status Selecting this options allows you to search for orders that are either Not posted which is the default setting, Posted or All.
  • Store
  • Posted dated
  • Created date
  • Description

Select a filter type for each search filter line.

Check the box to use the search criteria.
  • Begins with
  • Contains
  • Begins with: Selecting this option allows you to search for the beginning of a value in the field Description and. (This is the default setting) .
  • Contains: Selecting this option allows you to search for the number or letters in field Description. This field is only visible for types that allow free text search.
Fields & buttons Description
ID Displays stock counting journal ID.
Description Displays stock counting journal description.
Store Displays store description.
Created date Displays date when stock counting journal was created.
Status Displays status of the stock counting journal.
Posted date Displays date when stock counting journal was posted.
Next Click this button to review and edit the settings for the new stock counting journal before it is created.
