Goods Receiving Document Line Dialog

Inventory > Purchase > Receiving > Highlight goods receiving document line and select Edit button > Select ADD or Edit button

What is the dialog used for?

  • Add or edit a Goods receiving document line.
Fields & buttons Description
Barcode search Scan or enter the item barcode.
Item Select item from the list.
Variant description Variant description is displayed if the item is a variant
Unit Select which unit of the item is being received.
Received date Displays the expected receiving date but can be edited.
Ordered quantity When an item has been selected this field displays the ordered quantity of that item.
Received quantity Enter the received quantity.
Fields & buttons Description
Create another Check this box to keep the Goods receiving document line dialog open and ready to add the next item after you press OK. You can always click the cancel button to discontinue.
