Parked inventory view

Inventory > Journals> Parked > Edit a parked inventory journal

What is the view used for?

  • To view, edit and moved stock from main inventory to parked and back.
  • Parked inventory movements are negative for inventory removed inventory and into parked but positive to move stock back into main inventory.
  • The list of parked inventory lines can be filtered and searched.
Items

Search filter

Parked inventory lines can be searched for in a filtered search. One filtering line is added for each type of filter. You can even save your preferred filtering settings and they will be available when you open any of the journal line views.

 
Fields & buttons Description
Status Selecting this options allows you to search for orders that are either All Statuses which is the default setting, Openor Posted.

Description

Enter a test to search for in the description of the parked item

  • Begins with: Selecting this option allows you to search for the beginning of a value in the field Description and. (This is the default setting) .
  • Contains: Selecting this option allows you to search for the number or letters in field Description. This field is only visible for types that allow free text search.
Variant

Enter a test to search for in the variant description of the parked item

  • Begins with: Selecting this option allows you to search for the beginning of a value in the field Variant and. (This is the default setting) .
  • Contains: Selecting this option allows you to search for the number or letters in field Variant. This field is only visible for types that allow free text search.
Quantity

Enter a number to search for an parked inventory line based on the parked quantity

  • Equals: Finds all parked inventory with this exact quantity adjusted
  • Greater than: Finds all parked inventory which have a larger quantity adjusted.
  • Lower than: Finds all parked inventory with lower quantities.
Unit Search for parked inventory based on the unit of the parked item.
Reason code Search for parked inventory based on the reason code.
Posted date Search for parked inventory based on the date the parking was done.
Staff Search for parked inventory based on the user that parked the inventory
Area Not applicable to parked inventory.

Press this button to add search criteria and narrow the filter.

The following actions are available from a drop down menu when this button is pressed.

  • Save as default layout: The search lines you have selected will be saved to the view and opened when this view is opened.
  • Load default layout: If you have edited the search settings since you last opened the Retail items view select this option to reload the saved layout.

List of adjustments

Fields & buttons Description
Status (as a "dot" image)

Displays reservation status:

  • Open - Inventory has been parked
  • Posted - Parked inventory has been moved back into main inventory.
ID Displays the ID of the item being parked.
Description Displays the description of the parked item
Variant If the item is a variant item this field displays the variant description of the parked item.
Quantity Displays the quantity of items being parked. Negative quantity is used for stock being parked from main inventory and positive quantity is used to move items back to main inventory.
Unit Displays which unit of measure of the parked item has.
Reason Displays the reason code selected for this parked inventory line.
Posted Displays the date the item parking was done.
Staff Displays the staff that parked the item.
Area Not applicable to parked inventory.
Edit The parked inventory lines cannot be edited and therefore this button is disabled.
Add Press this button to create parked inventory journal lines to move inventory from main inventory to parked.
Delete The parked inventory lines cannot be deleted and therefore this button is disabled.
Button:
Move to main inventory
 Press this button to move all or part of the items in the selected parked inventory journal to main inventory.
Summary
Fields & buttons Description
Status Displays the status of the parked inventory journal. Parked inventory journals have the status open until they are archived.
ID Displays the ID of the parked inventory journal
Description Displays the description of the parked inventory journal
Store Displays which store the parked inventory journal is for.
