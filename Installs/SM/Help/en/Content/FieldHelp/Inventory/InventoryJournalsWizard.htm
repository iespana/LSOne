New inventory journal

Inventory > Journals> Journals

This wizard menu gives the user different options to create new stock counting journal and also a possibility to manage them.

What is the dialog used for?

  • Select the type of inventory journal to be created

Buttons Description
Inventory adjustment Create a new inventory adjustment journal. With inventory adjustments you can adjust the inventory levels of certain items for an example if something breaks a negative adjustment can be made.
Stock reservation Create a new stock reservation journal. Stock can be moved from main inventory to reserved. Customer orders can also reserve stock.
Parked inventory Create a new parked inventory journal. Parked inventory are the items that have been returned but for some reason can not be returned to the main inventory.

 

Next page in wizard

Fields Description
Description Enter the description of the new inventory journal.
Store Select which store the new inventory journal belongs to.

 

Permissions

  • Inventory adjustments
    • Manage inventory adjustments for all stores - allows creating inventory adjustment for other stores than the current one
    • Manage inventory adjustments - allows creating inventory adjustment only for current store
  • Stock reservation

    • Manage stock reservations for all stores - allows creating stock reservation for other stores than the current one

    • Manage stock reservations - allows creating stock reservation only for the current store
  • Parked inventory

    • Manage parked inventory for all stores - allows creating parked inventory for other stores than the current one

    • Manage parked inventory - allows creating a parked inventory only for the current store
