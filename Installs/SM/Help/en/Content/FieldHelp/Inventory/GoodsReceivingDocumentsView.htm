Goods Receiving Documents View

Inventory > Purchase > Receiving

What is the view used for?

  • View the Goods receiving documents on the system.

Goods receiving documents can be searched in a filtered search.

Search filter

Goods receiving documents can be searched for in a filtered search. One filtering line is added for each type of filter. You can even save your preferred filtering settings and they will be available when you open the Goods receiving documents view.

 

Fields & buttons Description
  • Status
 Selecting this options allows you to search for orders that are either Active which is the default setting, Posted or All.
  • Status
  • Posted date
  • Created date
  • Description
  • Vendor
  • Store

Select a filter type for each search filter line.

Check the box to use the search criteria.
  • Active
  • Posted
  • Begins with
  • Contains
  • Active: Selecting this options allows you to search for documents that are active.
  • Posted: Selecting this options allows you to search for documents that have been posted.
  • Begins with: Selecting this option allows you to search for the beginning of a value in the fields Description. (This is the default setting)
  • Contains: Selecting this option allows you to search for the number or letters in fields Description. This field is only visible for types that allow free text search.
Search field Enter the search criteria. For description and variant number you can enter a free text.

If there are more than one line press this button to delete a search criteria.

You can also unselect the checkbox to exclude a criteria from the search.

Press this button to add search criteria and narrow the filter.

The following actions are available from a drop down menu when this button is pressed.

  • Save as default layout: The search lines you have selected will be saved to the view and opened when this view is opened.
  • Load default layout: If you have edited the search settings since you last opened the view select this option to reload the saved layout.
  • Reset: Select this option to reset the layout to be as displayed in image above.
  • Toggle button locations: Select this option to set the and buttons in front of the filter type.
Button:
Search		 Press this button or hit enter button on keyboard to activate a search with the filters set.

Field and button description for the list view

Fields & buttons Description
ID Displays Goods receiving document ID.
Description Displays Goods receiving document description.
Vendor Displays vendor.
Store Displays the store's description.
Status

Displays document status:

  • Posted
  • Active
Ordered Displays number of items ordered.
Received Displays number of items received.
Created Displays the date when the purchase order was created.
Posted Displays the date when the Goods receiving document was posted.
Edit Highlight a line and select the edit button to edit the post.
Add Select add to add a Goods receiving document.
Delete Highlight a line and select the delete button to delete the line.
