Form profiles

A form profile is a collection of form layouts of different form types. These profiles can be added to stores.

Form profiles are created in Site Manager to be read by the LS POS which then uses this information to select the form layouts for printing.

To open Form profiles, go to Store setup > Profiles > Form profiles

Form profile card:

A default form profile comes with the system and cannot be deleted. If a store does not have a designated form profile, the default form profile is used.

To add a form profile, the upper button is clicked. A new form profile can copy from an existing profile.

To add lines to the profile, the lower button is clicked. First, form type is selected. Then, layouts of that type can be selected.

If no layout of the desired type exists, it can be added in Store setup > Forms and labels > Form layouts.

One form profile can be linked to more than one store.
