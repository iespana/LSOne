Trade Agreements: Line Discount

Trade agreements Line discount is a discount for items given automatically in the POS when the item is sold as long as the line discount’s restrictions are met.

Line discount can be set up from the item card or the customer card.  Line discount setups are a combination of item and customer/group/all customers.

To set up Trade agreement Line discount for an item:

  • In the ribbon go to Item > Items;
  • Select the item and open the retail item card;
  • Open the Discounts tab and select the Line discounts tab
  • Use the add button to add a new Line discount rule:
  • It is possible to have many rules for each item as long as they have different restrictions.
  • When the LS POS searches for a Line discount rule then it first looks to see whether a rule exists for each individual entry, then for each group and finally for all.
  • For one sales line it is possible that the LS POS may find many line discount rules. This is explained further here Trade Agreement Discount Configurations

Line Discount calculation at POS

The POS is by default set to calculate all discounts as soon as the items are sold. But if your store has a big average basket size the discount calculations can cause the POS to be less responsive. It is therefore recommended to change the discount calculation settings in this case.

To change the setting go to Tools > Administration options in the ribbon and select the Discount settings tab.

In the Customer discount fields select Calculate on total. Next time you log into the LS One POS the customer discounts will only be calculated when the Display total button is pressed.

Example - Line discount

Example of Line discount setup for an item:

  1. In the Site Manager ribbon select Item > Items;
  2. Open the retail item card for item Briefcase, Vinyl, ID is 50010;
  3. Open the Discounts tab and select the Line discounts tab
  4. Click the add button to set up the Line discount rule;
  5. Fill out the fields for the rule and close the window.
  6. One Line Discount rule has been created:
  7. In the LS POS sell item Briefcase, Vinyl,  ID is 50010:
  8. Add customer Mr. Rene Depaul to the sale and Finish the sale with cash.
  9. The discount is also visible in the Customer view. In the Site Manager in the ribbon select Customers > Customers, edit customer Mr. Rene Depaul.
