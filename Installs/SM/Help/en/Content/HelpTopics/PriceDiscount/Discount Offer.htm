Discount Offer

Discount offers are set up so that the LS One POS is able to sell items with automatic discounts when certain criterion is met. The discounted price is a discount off of the retail price.

When the LS One POS sells an item that triggers a discount offer:

  • The sale’s line item price is the discounted price;
  • An additional line is created below the item line with information about the discount offer.

The system can have an unlimited number of discount offers where each discount offer is set up separately. Each discount offer has a validation period and sets off offer lines that can be of several types, for example, item and retail groups.

To open Discount offers select Retail > Price/discount > Discount offers

Discount offer line dialog

You can set up one or many Discount offer lines per discount offer.

When the discount offer has been created and is ready, it needs to be activated.

Type: In this field, you can select what to add to the discount offer.

The types are the following

  • Item
  • Variant
  • Retail group
  • Retail department
  • All
  • Special group

When a group is added to a discount, only the Discount percent can be set. The price fields are not editable and are displayed with a zero.

Price/discount: The fields in this part are valid depending on the type of discount. When the Type is item or variant, more fields are available to edit.

The value for the discount is by default the same value as is set up on the Offers card, where it is possible to change it for individual offer lines.

 

LS POS Example: Discount Offers

This example uses the DAVI-LINE – 10% off discount offer.

  1. Before beginning this example make sure that the promotion offer DAVI-LINE – 10% off has the status Disabled.
  2. The DAVI-LINE – 10% off discount offer is valid for items in the special group DAVI line.
  3. Open Retail > Retail items > Retail items, to search for items in the special group, DAVI line.
  4. Go into the LS POS and sell item Black Dice Flow Watch Davi-s, use Item Search to find the item or enter Item ID 40210.
  5. The receipt panel should now look like this:

    No discount is given to the item since the offer has not been activated.
  6. Finish the sale.
  7. Go into the Site Manager and activate discount offer DAVI-LINE – 10% off.
  8. Go into the LS POS and sell item Black Dice Flow Watch Davi-s, use the Item Search to find the item or enter Item ID 40210.
  9. Finish the sale.

 

When the item is sold;

  • The line’s total is automatically changed to the discounted price;
  • An extra line is created that displays information about the triggered discount offer.
