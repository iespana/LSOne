Total Discount

Total discount is a trade agreement discount type which can be set up for customers.

How Total discount is set up and works is very similar to the Line discount setup.

To set up a Total discount for a customer:

  • Go to the Retail customer card;
  • Select action Total discounts in the Trade agreements box;
  • In the Total discount list use the Add button to create new rules;
  • It is possible to create many Total discount rules as long as they have different restrictions;
