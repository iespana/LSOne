Permission Groups

This is a list of all the permissions which are included in the LS One In-a-Box version and in which permission group each permission is included.

There are four permission group BasicAdmin, Store Manager, Shift Manager and Cashiers.

  • The BasicAdmin user group includes all permissions available in the LS One In-a-Box version. A user in this group should be used when setting up hardware and other initial settings
  • Store Manager can do almost everything the BasicAdmin can do. A user in this group should be used for all daily operations in the store.
  • Shift Manager has all the rights in the POS and can view some information and reports in the Site Manager.
  • Cashier can only operate the POS and cannot do some operations such as voids and returns.
