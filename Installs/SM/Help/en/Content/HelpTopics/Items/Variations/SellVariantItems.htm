Sell Variant Items

Variant items

Variant items can be sold on the LS One POS as any other item see Item Sale Operations

Header items

But if a header item is sold either by item search or item ID the POS displays a selection dialog where the POS user can select which variant item is being sold.

Note that the dialog displays only the attributes that exist as variant items. So if you select size S and the Size S does not come with Long sleeves the long button would disappear from the selection list.

The header item itself is actually never sold just the variant item which was selected.
