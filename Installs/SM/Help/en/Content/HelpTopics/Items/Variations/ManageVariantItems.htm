Manage Variant Items

When a new item is created and the item has different dimensions such as sizes or colors LS One creates one header item and each combination of dimension attributes is created as a variant item.

In the Items view the different types of items are marked with a icon in the front of each line:

  • retail items
  • variant headers
  • variant items

Each header item can be expanded to display the variant items it includes

The list of items can be flattened by selecting Show item variants check box under the search bar above the item list.

Edit the header item

The header item can be edited in the retail item view. To edit the header item you select the item from the list and press the button.

The header item properties that can be edited are

  • Item hierarchy
  • Units
  • Additional information
  • Images

Changes made to the above mentioned properties are not made to all the variant items. To edit the variant items properties you can use bulk editing

You can view and manage (edit and create) the following properties for all variant items at the header item

  • Prices and offers available to all variant items
  • Barcodes for all variant items

Bulk editing - Edit multiple variant items

If you need to edit all (or a set of) the variant items at the same time for an example if the base price changes you can use bulk editing.

In the Items view multiple items can be selected at the same time by holding the ctrl key at the keyboard while selecting the items lines.

When all lines have been selected you can press the button to edit the selected items all at once. The items do not have to be related to one another.

 

If all the variant items of the same attribute need to be changed at the same time you can use the Variant search option and filter the item list to only the items you want to edit. Then it is easy to select all the items from the list Ctrl + A and edit them.

Add dimensions and attributes and create new variant items

A new dimension can be added to a header item. You can also add attributes to existing dimensions.

Add dimensions or attributes:

  1. open the header item in the item view
  2. Go to the Variant items tab
  3. Press the Edit dimensions button
  4. In the Edit dimensions dialog create dimensions and/or attributes as needed. This is done the same way as described in steps 2 and 3 in Create Variant Item
  5. Press the OK button
  6. No variant new variant items have been created yet. Press the button below the variant item list to open the Create variant items dialog.
  7. This dialog displays all combinations of dimension attributes that do not exist as variant items. Select the combinations to be created and press OK

Now new variant items have been created for the new dimension and/or attributes.

Note that it is not possible to edit dimensions and attributes previously created. It is only possible to edit the dimensions and attributes that you are adding and after you press OK those dimensions and attributes become uneditable also.

Dimensions can not be deleted but when deleting attributes all the items using those attributes will be deleted.
