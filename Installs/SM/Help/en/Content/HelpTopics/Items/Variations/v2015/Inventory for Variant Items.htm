Inventory for Variant Items

The system keeps information for items depending on their variations setting.

There is a tab called Inventory on the retail item card. This tab displays stock information from each store and each available variation possibility.

 

This topic applies to version 2015.1 and older versions of LS One. If you are running a later version please read this.
