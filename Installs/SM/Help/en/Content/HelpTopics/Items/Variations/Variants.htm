Variants

Items can be sold in many ways. A typical item such as milk, which is sold by item bar code or set up on a button, does not need to provide any additional information about its features.

Items such as jackets, trousers or skirts can be sold in different colors and sizes, boots have different width and hardware such as screws have different length. We call the color, size, width and length dimensions. Each item can have multiple dimensions.

The dimensions have different attributes. The attributes for the dimension color could for an example be: Red, Yellow and Blue and the attributes for sizes, small, medium and large.

In LS One we use the dimensions for items that are sold in different variations. One header item is created with features to control which ever dimensions and attributes the item has. For each combination of dimension attributes will be created a variant item.

This chapter will show you step by step how to set up dimensions and attributes, how to create and configure header items and variant items and finally how this item is sold on the LS One POS.

Example

The fashion store just got a new jacket and needs to add it to the list of items. The jacket comes in three different colors, has different sizes and sleeve length.

When the item is created you enter the dimensions to the item the dimensions would be the following

  • Color
  • Size
  • Sleeve

Each of these dimensions have several attributes which also are entered when you create the item.

  • Color: Red, Black, Grey
  • Size: S, M, L, XL
  • Sleeve: Regular, long

When all the dimensions and variants have been created you select which combinations exist and which variant items will be created. If all combinations are selected 24 variant items will be created.

The variant items can be sold by ID, bar codes with button sales or by selling the Header item and selecting which variant item is being sold.
