Manage Retail Items

To open Retail items select Item > Items:

The Retail items list view:

In the Retail items view some actions are available for multiple items at the same time:

  • Price and shelf labels can be printed
  • Images can be imported for many items
  • Items can be imported or exported to excel templates

Item search

Filter the list of items with the item search.

For further information on how the item search works see Search filtering in the Items view field help.

Edit an item

To open the Retail item card, either double-click on the item line or select the item and use the edit button.

The Retail item card allows you to edit your item.

See the retail item view field help to get information about the functionality of each setting.

Bulk editing - Editing multiple items at the same time

If you need to change a set of properties on multiple items to the same value you can use bulk editing to edit multiple items at the same time.

For an example if a set of items need to be moved between tax groups this can be done in one action.

  1. Go to the items view: Retail > Retail items
  2. Select multiple items can be selected at the same time by holding the ctrl key at the keyboard while selecting the items lines.

    Use the search bar at the top of the list to filter the list of items to only the once you need to edit then it is easy to select all the items from the list Ctrl + A and edit them.

  3. When all lines have been selected you can press the button to edit the selected items all at once. The items do not have to be related to one another.
  4. In the Item view edit all the properties that need to be edited and press save.

 

In bulk editing:

  • All the items being edited are listed at the top of the item view
  • All changed fields are marked with green
  • You can revert your changes to one field with the revert field option
  • You can revert all your changes with the revert options

 

Editing multiple items in Excel

Item settings can also be changed by exporting a list of items to an excel sheet. Editing the settings in the excel document and then import the items again.

  1. Open the retail items view.
  2. Filter the list of items to the items you want to edit.
  3. Press Export... button and select List to Excel template... or List to Excel simple template...
  4. Enter a name for the excel file and browse to a location where to save the document, then press Save.
  5. Edit the information in the Excel sheet and save the changes. Here is detailed information about all fields in the excel document.
  6. In the retail item view press the Import... button.
  7. Select From excel document...
  8. Find and select the excel document you have created and edited and press open.
  9. In the Excel document for the retail items field select Merge. Information on import settings.
  10. Press OK and your item settings will be updated.
