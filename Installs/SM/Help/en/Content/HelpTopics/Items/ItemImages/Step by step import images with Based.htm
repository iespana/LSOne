6.2 Step by step, import images with Based on a data file

 

  1. Save your images with any name formatted in .jpg or .png.

  2. Create a text file with a full relative path for each of the images you want to import: item number; image full relative path

If the text file you are using exists in the folder you do not need to include the folder name. You need to create a line for each image in the text file.

  1. Go to Site Manager > Retail > Retail items > Retail items
  2. Click on Import images link in the Actions box.
  3. Choose your Image alteration settings.
  4. Select Based on data file > select the … button > choose the text file you are using > select Import button on the Import images dialog.


The Progress area shows you the progress of the import, how many images are successfully imported, how many have failed to import and how many files in the folder were skipped in the import.
