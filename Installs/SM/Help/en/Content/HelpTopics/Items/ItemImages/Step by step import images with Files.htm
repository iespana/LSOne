5.2 Step by step, import images with Files in a folder

  1. Save your images with the ID number of the item as the image name, with .jpg or .png. in one folder.
  2. Go to Site Manager > Retail > Retail items > Retail items
  3. Click on Import images link in the Actions box.
  4. Choose your Image alteration settings.
  5. Select File in folder > select the … button > choose the folder where you have saved your images > select Import button on the Import images dialog.

Note:
The Progress area shows you the progress of the import, how many images are successfully imported, how many have failed to import and how many files in the folder were skipped in the import.
