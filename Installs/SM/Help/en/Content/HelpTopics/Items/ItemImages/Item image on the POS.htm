Item image on the POS

The image appears in the item image area on the POS when the item is added to a sale.

Image: Item image on the POS

How to add the Item Image Area to the POS Touch Button Layout

On the LS One POS
  1. Log in to POS
  2. Find an empty button or change the functionality of some button (remember to change back to the original operation)
  3. Right-click this button and select button properties
  4. In the Button properties dialog select the Design mode enable operation.
  5. Press Ok
  6. Now press the button
  7. Drag and drop itemImageitem part into the Touch button layout
  8. Close the Customization dialog
  9. Press Yes to save the layout
In the Site Manager
  1. Log in to Site Manager
  2. Select Store setup > Look and feel > Touch button layouts > Select touch button layout
  3. Click on Change design
  4. Drag and drop itemImageitem part into the Touch button layout
  5. Click on Save
  6. Log out and back in to the POS
