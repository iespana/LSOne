Images

This section contains information on how images can be added to items and displayed in the POS at the time of sales and in the item search.

  • How to view the item image in the Site Manager
  • How to view the item image in the LS One POS

 

There are three ways to import images:

  1. Files in a folder: import multiple images from one folder;
  2. Based on a data file: use a text file to import multiple images from multiple locations;
  3. Add a single image through the Retail item view using the Add button.

Using the first two methods also allows you to resize and compress images.
