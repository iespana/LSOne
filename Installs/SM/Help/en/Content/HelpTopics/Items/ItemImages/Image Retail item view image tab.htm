8.3 Image: Retail item view, image tab

 

Add button

Select the Add button in this view to open the folders on your computer and add an image file to the item.

Delete button

Select the Delete button in this view to delete the image on display.

Image files on computer

Navigate to the correct image file and confirm your selection. Note that the image file name in this case does not matter. However, with this method you can only add one image at a time.
