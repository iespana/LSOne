User permissions needed for customer orders

The user permissions involved in customer orders are the following:

  1. Manage customer order settings - This permission controls whether or not you are able to view customer order settings in Site manager
  2. Manage customer order - This permission controls whether or not you are able to view a list of customer orders in Site manager
  3. Customer order - This permission controls whether or not you are able to create a customer order in POS
  4. Customer order management - This permission controls whether or not you are able to recall a customer order in POS
