How to cancel a customer order

Cancellation of a new customer order

New customer order can be cancelled by clicking the Void transaction button (Void transaction).

Cancellation of a recalled customer order
  1. Recall a customer order that has already been created by clicking the Recall customer order button (Recall a customer order) and select the order to be cancelled.
  2. If there are no payments on the transaction click the Void transaction button otherwise the payments needs to be voided before the transaction can be voided (Void transaction).

 

If an item is fully received, selected for pick up or partially received it cannot be voided. If the item has linked items then they need to be voided first.

Voiding an item on a new customer order

Item on a new customer order can be voided by pressing the Void item button (Void item)

Voiding an item on a recalled customer order
  1. Recall a customer order that has already been created by clicking the Recall customer order button (Recall a customer order) and select the order.
  2. Select the item to be voided and press the Void item button (Void item).

 

  • If other items are on the customer order the deposit on the voided item will be distributed to the other items displayed.
  • If other items are on the customer order and the total of those items is less than the voided item the deposit on the voided item will be used to fully pay the items on the order.
Voiding an item on a recalled customer order and pay back deposit
  1. Recall a customer order that has a deposit on it by clicking the Recall customer order button (Recall a customer order) and select the order.
  2. Void all the items (Void item).
  3. Click any payment button.
  4. Select Pay back deposit.
