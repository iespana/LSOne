How to pick up a customer order

Customer order can be picked up partially or entirely.

Full pick up of customer order

 

  1. In the POS Recall a customer order that has already been created by pressing the Recall customer orders button (How to recall customer orders).
  2. Select the customer order to be recalled
  3. Click a payment button
  4. Customer order action dialog is displayed, select Pick up entire order
  5. Payment dialog with the total amount to be paid for the items selected minus the previously paid deposit is displayed
  6. Finish the transaction
Partial pick up of customer order

 

  1. In the POS Recall a customer order that has already been created by pressing the Recall customer orders button (How to recall customer orders).
  2. Select the customer order to be recalled
  3. Click a payment button
  4. Customer order action dialog is displayed, select Partial pick up of items
  5. Customer order details dialog is displayed
  6. Select items for pick up
  7. Click Finish
  8. Payment dialog with the total amount to be paid for the items selected minus the previously paid deposit is displayed
  9. Finish the transaction
