4.3.3 The Log Tab

The Log tab show all messages created by the Scheduler back-end.

 

Figure 21: The Job Log tab

Enter or select the Date range to filter the messages shown and click the Search button to show messages within the filter. The messages shown include the date and time when the job was run and any error messages encountered.

 

4.3.4 Running a job

To run a job, go to Home, General setup, Replication, View jobs. Select the job you want to run and click the edit button. Set the Source location and Destination location. Make sure your job has subjobs.

 

Figure 22: The Run job button

Then click the Run job button to run your job.
