Import from Excel Document

To import the excel document select in the ribbon Tools > Excel document

Selecting this option will bring up a dialog to choose your Excel document.

Note: It is very important that this can only be a document that was based on the LS One Excel import template that the Site Manager creates.

Once a document has been selected, the Site Manager will ask for details on how the import should be done for each set of data.

By default, the Merge option is selected for all the data sets.

Field Calculate profit margins: Check if the import process should calculate the profit margins for the retail items.

Available import options:

  • Insert if not exists: Only attempt to insert records if they do not exist in the Site Manager.
  • Merge: Does a smart merge if a record exists in the Site Manager. Existing records rule over an empty field from Excel.
  • Override if exists: Overrides existing records if they already exist in the Site Manager.
  • Ignore: Ignores the table and imports nothing.

After the OK button is pressed the Site Manager will display an import report, where explanations are listed if something could not be imported.
