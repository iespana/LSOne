Importing Data

The Site Manager has an operation to import data from an Excel document for a few types of data. The Excel import can be used either to insert new data or to update existing data.

The data which can be imported into the Site Manager from an Excel document is:

  • Retail items
  • Retail groups
  • Retail departments
  • Customers
  • Vendors

The Excel file has to have a certain layout and in the Site Manager there is an operation to create a template excel file.

The process for importing data:

  1. Make sure that the conditions in Site Manager for the excel import are fulfilled.
  2. Create a new Excel import template to enter the new items in.
  3. Input the data into the excel template sheet document.
  4. Use Site Manager’s operation Import from Excel document to import the excel sheet.

The process is further explained in the sub-chapters of this book.
