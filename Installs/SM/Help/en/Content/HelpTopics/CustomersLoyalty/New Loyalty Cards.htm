New Loyalty Card(s)

Retail > Loyalty > New loyalty card(s)

Create a new loyalty card by entering the information in the New loyalty card(s) dialog.

You can create multiple cards at once and add the customer to them later. This is useful if you are offering loyalty cards to be created on the POS. Then you can simply use the operation “Add customer to Loyalty card” (see Loyalty on the POS).

Select a scheme from the menu or use the add button to create a new one, or the Edit button to edit the scheme selected (opens Loyalty scheme dialog).

A customer must be added either now or later for the card to be issued and valid for use. This can be seen in Loyalty Transactions view (see chapter Loyalty Transactions ).

The card type (how the loyalty card should be used) must be chosen:

  • Card tender: The card can be used for accruing and spending points
  • No tender: The card cannot be used as tender in a POS transaction, but can be used for accruing points and for other kinds of uses such as to access online deals.
  • Contact type: If a customer has multiple cards, this card gathers the points information from all the loyalty cards registered to the customer and allows the customer to use them all together.
  • Blocked: The card is blocked for use by the customer. *This cannot be used for new loyalty cards, only for editing loyalty cards in Edit loyalty card dialog in the Loyalty cards view.
  • Scheme ID: Select a scheme that has already been created.
