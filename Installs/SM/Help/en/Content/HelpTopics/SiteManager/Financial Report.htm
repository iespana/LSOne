Financial Report

To open the Financial report select Retail > End of day > Financial report

To print a financial report:

  • Select a date range for the Financial report in the Start date and End date fields;
  • Select the store/stores to create the report for;
  • Select the terminal/terminals to create the report for;
  • Select the View report button
