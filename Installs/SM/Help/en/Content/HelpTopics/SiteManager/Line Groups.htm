Mix and Match Line Groups

For each Mix and match offer you need to create new line group(s). The line group decides how many units of the items assigned to it need to be sold to trigger the offer. You can put many items in the same line group or just one. You can also set up many line groups in the same Mix and match offer, allowing you to combine quantities in the offer.

Line groups:

  • Must be created for the Mix and match offer before the offer’s lines are created;
  • Are created for each Mix and match offer;
  • Are created with the button;
  • When the line groups have been created they can be selected for the mix and match offer lines.

The following are some examples in the Demo Data:

 

Example: Diary x 2 + Stapler

 

This offer has Number of lines to trigger set as 3 which means that three sales lines are needed for the offer to be triggered. The sales items that are included depends on the definition of the mix and match lines.

This example has two line groups, A2 and B1. Two lines are needed for A2 and one line for B1.

Line groups card:

Each Mix and match line belongs to a line group.

The setup and criterion of a Line group:

  • There are two line groups with IDs A2 and B1.
  • The setup requires three items to be sold in total, two for line group A2 and one for line group B1.
  • To trigger this offer sell one Stapler and two Appointment diaries.
Example: Drinks – least expensive free

This Mix and match offer will be triggered when any three items of the possible items are sold in whatever combination, it could even be three of the same item. When the sale fulfills the offer’s criteria the mix and match offer is triggered and the least expensive item is free. The discount, which equals the price of the least expensive item, is then proportionally distributed between all of the lines.

This offer only has one line group, A3.

A3 is set up with Number of lines needed as 3.

All the offer lines are marked with Line group A3.

The setup and criteria:

  • This offer only has one line group, A3.
  • Three items need to be sold for this line group.
  • Since the items for this offer are all in the same line group then any combinations of these items trigger the offer as long as three items are sold.
Example: Buy 3 – least expensive free

This mix and match offer is only triggered when each item in the offer is sold once. When the sale fulfills this criterion then the least expensive item is free. The discount, which equals the price of the least expensive item, is then proportionally distributed to all of the lines.

Setup of the line groups:

The setup and criteria:

  • The line groups are three numbers: A1, B1 and C1.
  • The setup requires one item to be sold per line group.
  • The item lines in the offer are set up so that each item has its own line group.
  • The offer is only triggered when one item per line group is sold, that is three items in total.
