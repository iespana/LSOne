New POS User

To create a new POS user, select Security > POS Users > New POS User

  • ID: Max 20 digits, a numeric field
  • Description: Usually the POS user’s name
  • Password: The POS user’s password, a numeric field
  • Retype password: Retype the previously entered password
  • Next login:Check this field if the user has to change their password next time they log onto the POS
  • Copy from:Optional to use. Select a user whose permissions can be copied over to the new user

When OK is selected, the POS user is created and the user card opens where more information for the user can be configured.
