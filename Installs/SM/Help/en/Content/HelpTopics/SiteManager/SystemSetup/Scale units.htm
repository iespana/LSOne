Scale units

To fulfill the Scale Certification requirements, these four scale units have to be selected from the units list for the scale to work properly.

If the user tries to weigh an item that does not have one of these units, the POS will not allow the weighing.

This functionality will only be used when the scale has been activated on the hardware profile.
