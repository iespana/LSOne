Administrative options

Part of General setup is Administrative options.  In Administrative options different things for the company and stores are set up and configured. The setup in Administrative options should be one of the first tasks before the data for the company is inserted. 

To open Administrative options select General setup > System > Administrative options

The quickest way to open Administrative options is to use the ribbon Tools > Options

Administrative options:

The administrative options are grouped on fourteen tabs, which are detailed further in the following sub chapters:

  • Localization
  • Security
  • Active price/discount
  • Discount settings
  • Scale units
  • Number sequences
  • Web service license
  • Store management
  • Inventory settings
  • Loyalty
  • Customer ledger
  • Site Service
  • Decimal formats
  • Blank operations
