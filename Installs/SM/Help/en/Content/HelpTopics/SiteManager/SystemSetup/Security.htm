Security

Tab Security:

 

Logins: Fields to control the logins for users.

Active Directory: Two fields: Enabled and Domain. See more information on how to create a new active directory user.

Auditing: Either the auditing is On or Off. Auditing registers changes made to the data, who changed it and when.

When auditing is on, all of its data is stored in a separate database. The auditing database has the database name with _Audit added to it.

 

See more information on Security tab - Administration Options view.
