Terminal card:

The terminal card has a few fields in the top part of the window and then most of the configuration is grouped into seven tabs.

When a terminal has the same setting as the store then the terminal is superior to the store, and therefore all settings on the terminal will overrule the store settings.

The profiles and touch button setup configure different functions for the POS terminal. Each profile can be set up for many terminals.
