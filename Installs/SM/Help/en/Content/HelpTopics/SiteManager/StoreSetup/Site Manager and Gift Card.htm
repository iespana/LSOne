Site Manager and Gift Card

All activity regarding gift cards is registered in the Site Manager. The operation to view and work with gift cards in the Site Manager is Gift cards.

To open Gift cards select Retail > Vouchers > Gift cards

The Gift cards list:

With this operation the user can:

  • Issue new gift cards by using the Add operation. The gift card’s amount and currency needs to be entered. It is possible to use the count field to issue many gift cards at the same time. Issuing gift cards in the Site Manager is only valid for gift card option ID; the other options issue the gift cards in the LS POS.

  • Search for issued gift cards.
  • Activate existing gift cards, when gift cards are created here they are by default inactive. Gift cards have to be inactive to be sold on the POS. A gift card which is sold on the POS changes its status to active.
  • Delete gift cards.
  • Edit gift cards to view the card’s usage log.
