Create a Suspension Type

Suspension types are created to capture information such as free text input, dates or customer while suspending transaction at the POS.

To create a new suspension type follow these steps:

  1. Go to Sales > Suspended > Suspended types

  2. Suspension types view: Press the button to create a new suspension type.

  3. Enter a description for the suspension type and select if End of day should be allowed with transactions suspended on this suspension type. This setting can be overridden at the store or terminal level by selecting Store default or Terminal default.

  4. Press OK
  5. Suspension type view: To add fields for gathering information at the time of suspension press the button.
  6. Enter a label for the field in the description for the new suspension field
  7. Select a type, there are different types to select between:
    1. Text - Asks the user to enter a free text.
    2. Customer - Asks the user to select a customer from the list of registered customer.
    3. Name - Asks the user to enter a name of the customer or user that is suspending the transaction
    4. Address - Asks the user to enter an address. All the address fields are available in the form.
    5. Infocode - Select an infocode that will pop up when a transaction is suspended on this type.
    6. Date - Asks the user to select a date from a date picker.
  8. Decide if the field you are creating should require an input or be optional. If the field is required the OK button will not be available until the field has been filled in.
  9. Press OK

 

Repeat steps 5 - 9 to create as many field as needed.
