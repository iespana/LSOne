Payment Types

To open Payment types select Setup > Payments > Payment types

You can create all possible payment types for all stores in the system here.

These are the payment types in the Demo Data:

The payment type has:

  • An ID, automatically given from a number series;
  • A description;
  • A default function;
  • Limitation options - optional to use.

With every payment type it is possible to set up a limitation. Limitation is ideal to use for payment types that are not allowed to be used for every item, item group or store.
