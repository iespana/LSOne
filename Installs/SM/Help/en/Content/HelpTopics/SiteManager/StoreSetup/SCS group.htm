Size Groups, Color Groups and Style Groups

The basic data has a list of all possible sizes, colors and styles. Within the groups, possibilities can be grouped together in as many possible combinations as needed. There are no limitations on how many groups the basic data can exist in. For example, the color blue can be in many color groups.

When all of the possible sizes in the company are created, the list of entries can become very long. The size group allows you to gather together the sizes that items are available in. For example, children’s clothes may be available in XS, S, M, L and XXL and not in sizes 40, 42 and so on. You can create a size group that is specifically for children’s clothes to include only the relevant sizes.

The same logic is true for the other groups: Style and Color.

The groups are then used for the items to define the possible variations per item. This is to simplify the work in creating variations per item.

How to open Size groups
  • Select General setup > Variations > Size groups

 

How to open Color groups
  • Select General setup > Variations > Color groups

 

 

How to open Style groups
  • Select General setup > Variations > Style groups
