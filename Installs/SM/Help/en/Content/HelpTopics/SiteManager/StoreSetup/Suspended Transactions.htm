Suspended Transactions

In the LS POS it is possible to suspend and recall transactions.

Each terminal can suspend and recall its own transactions by default. For suspended transactions to be recalled from other terminals within the store the LS One Site Service needs to be installed and configured. More information about the Site Service.

In Site Manager it is possible to view all suspended transactions.

To view suspended transactions in Site Manager open Sales> Suspended > Suspended transactions:

  • The operation Recall in the LS POS works very similarly, it displays suspended transactions for the store and with it, it is possible to retrieve the suspended transaction into a sale and continue to work with it there.
  • The Site Manager’s Suspended transactions lists all suspended transactions for the company.
  • In the Site Manager’s Suspended transactions it is possible to filter the list by suspension type.
  • The selected suspended transactions can be deleted by using the delete button.
Permissions: Manage central suspension needs to be granted for the user to be able to access the view. Manage suspension settings controls whether the user can edit and or delete suspended transactions.
