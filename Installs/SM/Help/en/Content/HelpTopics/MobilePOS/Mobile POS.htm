LS Omni Mobile POS

LS One version 2017 can now connect to LS Retail's mobile POS. The LS Omni POS can run on iOS and Android devices and support a broad selection of retail operations.

To be able to connect the mobile POS to LS One you will need to setup an Omni server, you find more information on how to do that in the LS Omni Server installation guide.

Highlighted features of the LS Omni Mobile POS:

  • Sales by barcode or item lookup
  • Multiple payments
  • Returns
  • Manual discounts
  • Automatic discounts and promotions
  • Customers
  • Customer specific discounts and prices
  • Suspend transaction that can be recalled at any terminal
  • Receipt printing on printing station
  • Email receipt
