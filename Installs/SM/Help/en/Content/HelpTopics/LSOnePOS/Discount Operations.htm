Discount Operations

Items can be discounted both automatically and manually. This chapter explains the manual discounts. Manual discounts are found under button Discounts.

Note: You can set different maximum discounts for different users in the Site Manager. There is a pop-up window that returns an error if you try to give a discount that is too high. Some items do not allow discounts. This can be set up in the Site Manager.

Note: You cannot discount one item with a combination of percentages and amounts, if you do this, the POS will discount the last discount from the total due, not the discount that has already been made.

Line Discount Percent

An input window where the discount percentage is specified for the selected item.

The discount line appears below the item line.

Line Discount Amount

You can discount an amount from a selected item. Type the exact amount that you would like to discount into the pop-up window. The system calculates the amount to discount by percentage and displays it in the receipt panel.

Total Discount Percent

A discount percentage can be applied to all of the items in the transaction. When you have entered the percentage, a discount line appears below each item line in the transaction.

Total Discount Amount

An input box where the total discount amount can be specified for the current transaction. In the sale, press the Total Discount Amount button and enter the amount you would like to discount from the total sum of the transaction – including items that are sold after you have completed this operation.
