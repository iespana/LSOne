Activate a license for versions 2014 and later

The following procedure applies to LS One POS versions 2014 and later.

In order to activate a license, regardless of what type of license it is, the POS has to be connected to the internet and has to be able to access the LS One license service.

Demo license

When the LS One POS 2014 is started for the first time it has a demo license. The demo license will expire three months after the POS is first started.

The demo license is restricted and will not be able to perform the following operations:

  • End of day, End of shift, Print X, Print Z, Tender declaration, Clear quantity, Void item, Return item, Return transaction, Customer search
  • It will not be possible to sell more than 6 items.

 

Activate a license

To activate your license:

  1. Select the License button on the log in screen
  2. Enter the account name and password you have got from your license provider and press Next.
  3. Select which type of license you are going to use for this POS by selecting it from the list.
    Each license account can have many LS One licenses that either are perpetual or subscription based.

  4. Select any license from the list of your free licenses and press Next
  5. Enter a description for the POS you are collecting the license for and press Finish.
    It is easier for you to manage your licenses if you use this field wisely and enter some information about where this POS machine is located for an example.
  6. Now your LS One POS is fully licensed. If you need information about your license you can open the about information from the POS login screen Settings > About LS One.

 

Note:

  • A license can be deactivated the same way.
  • This has to be done for each POS machine in your setup.
