Item Sale Operations

Items can be sold in the LS One POS by:

Entering the item number

When the POS user knows the item number it can be keyed into the Item/bar code/quantity input panel. Just type the item number and then select enter.

Entering the bar code

The bar code of the item can be entered as the item number. Remember the bar code has to exist for an item to be recognized by the POS.

After you press enter, the item is added to the transaction and displayed in the receipt panel:

Using the Quick buttons

Quick buttons are buttons where a specific item has been assigned to the button. The POS can have many quick buttons.

On the POS, press the button Dairy, which is a group of many quick buttons for items. Press one or more of the quick buttons.

The Home button takes you back to the main options:

Scanning the bar code;

Scan the bar code and the item in the transaction is displayed on the receipt panel.

Using the Item Search to find items.

When Item Search is used, a new window appears. This allows you to search for an item by its number, name or image.The item search operation in POS has two parameters that can be set. The first one is search mode and the second is retail group.

The search mode parameter has five different settings:

  • Default: Search results are displayed as list, honoring the settings in Site Manager > Functionality profile > Items tab.
  • List: Search results are displayed as list.
  • List with price: Search results are displayed as list with an additional price column.
  • Image: Image and name is displayed for each item in search results.
  • Image with price: Image, name and price are displayed for each item in search results.

With the retail group parameter you can select which retail group you want to narrow the search to by default. This could for example be if you wanted to open an image search with only items from the retail group Vegetables. This parameter can be left blank if you want the search to display all items.

See Button Properties for more information on how to edit the properties of the buttons and changing operations.
