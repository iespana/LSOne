Customer card setup for a card reader

User can have his/her account linked to a customer card which can be swiped using a keyboard scanner, when doing a transaction in POS.

Example:

Customer card numbers are e.g. 8 characters and all start with the five digits 79244. If you would for example swipe a card number 79244005 then first of all the POS recognizes this as a card since the first five digits in the number scanned, matches bin series of type customer. Then it checks if there exists a customer having a customer card which number is 79244005.

In Site Manager create a new card type called Customer card by selecting General setup > Payments > Card types

  1. Select the Add button . A New card type dialog is displayed.
  2. Enter e.g. Customer card as description.
  3. Select type as Customer and press OK button.
  4. Select to edit the customer card type.
  5. Select to add a new card number series. Enter 79244 as the series number from and 79244 as the series number to. The POS will now recognize that swiped cards having numbers that start with 79244 are of type customer.

 

Select the Customer Card Dialog to link a customer to a card number e.g. 79244005.

You have now setup a card for a customer and are able to swipe the card with a keyboard scanner into the POS.

 

Hardware profile must be configured for a keyboard scanner before the user is able to swipe cards and it also needs to be setup for Site manager.
