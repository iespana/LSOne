Store Transfers

Using the LS Omni Mobile Inventory app stock can be transferred between stores.

Example

Store A has requested some items from Store B through a phone call. Store B fills in a store transfer order in the LS Omni Mobile Inventory app and sends it to store A.

Store A receives the items opens the app and can mark the transfer order as received.

 

Preparation

To be able to fill in the transfer order in the app first the transfer order needs to be created in the Site Manager.

The purchase order can either be filled in in the Site Manager or the transfer order can be left blank and can be filled in in the app.

You can create a black purchase orders to each of your stores to have them ready to fill in in the app and send them when needed.

 

How to send a transfer order

In this case a blank transfer order has been created in the Site Manager to send stock from the Supermarket to the Restaurant.

In the layout provided in the Omni data pack you find the Store transfers menu at the bottom of the startup menu. It includes the operations Sending and Receiving
If you have created your own button setup you need to add a button with the purchase worksheet operation assigned to it.

 

To fill in a transfer order and send it from the LS Omni Mobile Inventory app:

  1. The LS Omni Mobile Inventory app is connected as a terminal in the sending store, the supermarket.
  2. Go to Store transfers
  3. Select the Sending option
  4. Select which store to send the stock to. In our case it is the restaurant.
  5. Here you will see all transfer orders that have been created in the Site Manager. Some might be filled in already but others might be empty and can be filled in in the app.

    If you get the message No document found you first need to create the transfer order in the Site Manager.

    You also need to be sure to create the transfer order from the correct sending store and select the correct receiving store.

  6. Add items to the transfer order by searching by ID or description and selecting the quantity to be transferred.
  7. You can save the transfer order at anytime and continue working with it later on.
  8. When all items have been added to the transfer order and the transfer has left the building you press the send button. Then the transfer order will be ready for the receiving store, the restaurant, to receive the goods.

 

How to receive a transfer order

In this case a transfer order has been created in the Site Manager and sent from the Supermarket to the Restaurant.

In the layout provided in the Omni data pack you find the Store transfers menu at the bottom of the startup menu. It includes the operations Sending and Receiving
If you have created your own button setup you need to add a button with the purchase worksheet operation assigned to it.

 

To receive the transfer order in the LS Omni Mobile Inventory app:

  1. The LS Omni Mobile Inventory app is connected as a terminal in the receiving store, the restaurant
  2. Go to Store transfers
  3. Select the Receiving option
  4. Select the store you are receiving stock from.
  5. Here you will see all transfer orders that have been sent from this store to you and have not been received.
  6. Select the transfer order and open it.
  7. You can edit the lines of the transfer order if you did not receive the same quantity as the sending store stated.
  8. You can save the transfer order at any time and continue working with it later on.
  9. When you are finished updating the transfer order receiving quantities you press the Send button to update the on-hand inventory of the items.
