Setup for LS Omni Mobile Inventory app

To be able to use the LS Omni Mobile Inventory app with LS One there is some initial setup that needs to be done.

  1. The LS Omni Server has to up and running. (Installers and setup guides are to be found at the LS Retail Partner Portal)
  2. Then the Apps need to be installed.
  3. For using the app you will need to order and activate LS Omni Mobile Inventory app licenses.

 

 

Site Manager setup

Some setup is needed in the LS One back office.

  1. Omni functionality profile
  2. A terminal for the omni device
  3. The button menus for the LS Omni Mobile Inventory app need to be setup.

 

All the Site Manager setup can be imported in a default data package called Omni Data. This package include the functionality profile, terminal and button menus. Read here how to import a default data package.
