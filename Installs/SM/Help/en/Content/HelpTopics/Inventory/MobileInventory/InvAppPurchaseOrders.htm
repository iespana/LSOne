Purchase Order Menu

Purchase orders can be created using a mobile device like an mobile phone running iOS or Android. This enables the user to do the purchase ordering at place in the store or warehouse for an example.

Preparation:

To be able to create purchase orders in the LS Omni Mobile Inventory app you first need to do the following:

  1. Create a purchase order template for your store in the Site Manager.
    • The purchase order template has some settings that affect how the purchase ordering mechanism in the inventory app works.
  2. Open the purchase order worksheet in the Site Manager.
    • This will populate the purchase worksheet with the items from the template filter and calculate the quantities using the item replenishment settings.

How to work with purchase worksheets in the app:

In the layout provided in the Omni data pack you find the Purchase order menu on the startup menu. It includes the operation Purchase worksheet.
If you have created your own button setup you need to add a button with the purchase worksheet operation assigned to it.

 

  1. Under purchase worksheet you will find a list of all purchase worksheets for your site.
  2. Select the purchase worksheet to receive by pressing the line
  3. The purchase worksheet header details are displayed on the screen.
  4. Press open
  5. The purchase worksheet is displayed with all the item lines and their suggested quantity to be ordered.
  6. The quantity to be ordered can be edited by pressing the button on the right side of the line.
  7. Lines can also be added to the purchase worksheet by searching for it by description, ID or by scanning a barcode.
  8. At any time you can press the save button to Save your edits and continue later.
  9. When done with editing the list of items to be ordered and their quantities press the Send button.
  10. When the send button has been pressed the purchase worksheet has been updated and can be posted in the Site Manager to create the purchase order.
  11. When the purchase order has been created it can be received either in the LS Omni Mobile Inventory app or in the Site Manager goods receiving.
