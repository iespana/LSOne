Stock Counting Menu

Stock counting can be preformed using a mobile device like an mobile phone running iOS or Android. This enables the user to walk around the store or warehouse with their device and enter their count of each item.

Preparation:

To be able to do stock counting in the LS Omni Mobile Inventory app you first need to do the following:

  1. Create a stock counting template for your store in the Site Manager.
    • The stock counting template has some settings that affect how the stock counting mechanism in the inventory app works.

In the Omni data.xml default data pack there exists one stock counting template with four areas. See here how to import default data packs.

 

How to do stock counting in the app:

In the layout provided in the Omni data pack you find the Stock counting operation on the startup menu. If you have created your own button setup you need to add a button with the Stock counting operation assigned to it.

 

  1. When you press the Stock counting button in the inventory app you see a list of all stock counting templates.
  2. Select a stock counting template.
  3. If the stock counting template has areas then select an area to count for.
  4. When this is done you have come to the stock counting journal where you can scan item barcodes or search for them by ID or description.
  5. The item is added to your stock counting journal and depending on the quantity method in your stock counting template the app either sets a default quantity or requests you to insert your count.
    • Stock counting template setting: Quantity method;
      • Ask - The app requests the user to insert the counted quantity.
      • Quick scan - The item line is added to the journal with the default quantity also set in the stock counting template.
  6. If you have more than one area that should be counted in this worksheet you can go back one step and count for another area.
  7. When all items have been counted the stock counting journal is sent to the LS One Site Manager where the journal can be posted to update the inventory.
  8. The send option can be found on top of an open stock counting journal or in the list of stock counting template you have a button with action options for each template.
