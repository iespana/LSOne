How to create and send inventory transfer orders

Inventory transfer order

A store can create a transfer order of items that is to be sent to another store. This transfer can be based on a transfer request or not. Once the items are sent, the receiving store has to confirm the receipt of the items.

To create a transfer order:

  1. Select General setup > Inventory > Transfer orders
  2. Click the ADD button
  3. In the transfer order dialog select whether you like to create a transfer order from a request, from an empty transfer order or copy existing transfer order click the OK button.

 

  1. In the lower list view select the items and quantity you like to add to the order

 

A transfer order has now be created. To send the order press the Send button.

 

It's possible to add items to the order until it has been fetched by the other store.
