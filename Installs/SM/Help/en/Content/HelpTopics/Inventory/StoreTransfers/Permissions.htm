User Permissions

The user permissions involved in store transfer are the following:

  • View inventory transfer requests
  • Edit inventory transfer requests
  • View inventory transfer orders
  • Edit inventory transfer orders

Note: There is no permission for viewing inventory in transit as that is already limited to head office.
