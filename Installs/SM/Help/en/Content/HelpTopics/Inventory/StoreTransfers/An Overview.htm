An Overview: Transfer Orders and Requests

The Store transfer functionality can be split into two parts:

Inventory transfer requests

A store can send a request for items to another store (inventory transfer request). That store can then create a transfer (inventory transfer order) based on the request.

Inventory transfer orders

A store can create a transfer of items order that is to be sent to another store. This transfer can be based on a transfer request or not. Once the items are sent, the receiving store has to confirm the receipt of the items.

 

To explain inventory transfers, the documentation uses an example of a high inventory store *HI (Store B in scenario below) and a low inventory store *LI (Store A in scenario below).

This is the company setup that we are going to use in these scenarios:

 

A basic scenario

  1. Store A has a shortage of product X (HI) and store B has an excess of X (LI).
    • This is done manually or through the inventory lookup in the POS
  1. Store A sends an inventory transfer request to store B.
  2. Site Manager > General setup > Inventory > Transfer requests > Sending tab > Add button > [enter information] > Items in request area > Select Add button [add information on items] > Select Send Request button
  3. Store B receives the request and sends an inventory transfer order to Store A to confirm how much of product X it is going to send to store A.
  4. Site Manager > General setup > Inventory > Transfer orders > Sending tab > Add button > [From request] > Send request button
  5. Head Office monitors the sent order through Inventory in transit.
    • Site Manager > General setup > Inventory > Inventory in transit > Highlight line and select View Inventory Transfer button
  1. Product X is delivered to store A.
  2. Store A confirms the delivery.
    • Site Manager > General setup > Inventory > Transfer orders > Highlight order line > To receive tab > Receive button

Additional features

  1. Store A can add to the inventory transfer request until it is fetched by store B.
    If Fetched check box is clear:
    • Site Manager > General setup > Inventory > Transfer requests > Select Add button / Highlight order line and select Delete button
  1. Store B is closing for the weekend and has an excess of perishable goods. Store B creates an inventory transfer order which it sends to store A, which is open at the weekend.
    • Site Manager > General setup > Inventory > Transfer orders > Sending tab > Add button > [Create empty transfer] > Items in transfer area > Select Add button > [enter item information] > Send request button
