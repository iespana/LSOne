Stock Counting Template

Stock counting templates are used to create stock counting journals efficiently. The stock counting template includes settings on how the stock counting journal behaves during stock counting.

The stock counting templates contains settings both for stock counting journals in the Site Manager and in the LS Omni Mobile App.

How to create a stock counting template:

  1. Go to Inventory > General > Templates
  2. Press the add button to add a new template.
  3. Give the template a description and select Template type: Stock counting.
  4. Decide which store the template is available for or if it is available for all stores.
  5. Click OK

Item filter tab

In the item filter tab you can create a filter of items which are added to the stock counting journal created from the template.

The item filter is applied in the Site Manager and not in the LS Omni Mobile Inventory app where the items have to be added line by line.

General tab

In the general tab there are settings which apply to all types of inventory templates. Such as unit selection and what can be changed in each line.

Purchase worksheet tab

This tab has settings which are not used by the stock counting template

Stock counting tab

In this tab there are settings specific to the stock counting template and some settings which are specific to the LS Omni Mobile Inventory app.

The areas are only used in the LS Omni Inventory apps.

 

For detailed description of each field read the field help for the inventory templates view
