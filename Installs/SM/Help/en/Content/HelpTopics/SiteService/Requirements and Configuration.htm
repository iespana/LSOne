Requirements and Configuration

The Site Service needs to be installed and configured before it can be used in the system.

To have the LS Site Service up and running the following needs to be done:

  1. Install the Site Service. Information about how to install the Site Service is in the LS One Site Service Installation Guide.
  2. Create a Site Service profile and apply the connection parameters.
  3. Specify which Site Service functions should be active for the system.
  4. Assign the Site Service profile to the store and, or terminals.
  5. Set up Site Service parameters in Administrative options.
