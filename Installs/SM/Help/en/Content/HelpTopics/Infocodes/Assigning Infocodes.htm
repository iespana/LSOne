Assigning Infocodes

Infocodes can be assigned in the Site Manager to the following:

  • Retail items
  • Retail groups
  • Suspended transactions
  • Customers
  • Store’s payment types
  • Different actions in the functionality profile
  • Infocodes need to be created before they are assigned.
  • It is possible to assign more than one infocode for all the actions except actions in the functionality profile.
