Infocode Subcodes

The infocode’s subcodes are the selections the infocode displays when triggered in the LS POS.

The infocode’s subcodes are only for infocodes of the Selection list type and Selection buttons.

The infocode subcode card:

The field Description is an important field to fill out because this is the selection’s text displayed in the LS POS.

On the subcode’s card, another important field is the Trigger function.

This field has the following options:

  • None: When this value is used, the other fields below this field are unavailable. Option None is quite common to use when no other extra information is needed.
  • Item: Subcodes with this value open the other fields below this field. The Trigger code field is valid for the Retail item list. When this subcode is selected in the LS POS, information about the item is registered on the POS. Depending on the setup amount; the amount can update the item’s sales line.
  • Infocode: For this option the trigger code field opens the infocode list where an infocode can be selected.
  • Tax group: For this option the trigger code field opens in the tax group list where a tax group can be selected. The other fields are inactive.
