7.1 Central Table Functionality

The LS Site Service needs to run on one of the terminals or on a central computer in the restaurant. The other terminals will connect to the LS Site Service when tables are accessed or their status changes and sends messages to the other terminals so that they can set the same status on the tables on their screens.

When a user opens a table on one terminal, the LS Site Service will lock the table on the other terminals so they cannot be changed. A table is locked by the staff member that accessed it first. The staff handling the table can be changed by using the operation Change Staff.
