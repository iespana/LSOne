1.13 Hospitality in the Site Manager

Site Manager > General setup > Hospitality

 

The options in the Hospitality plugin include:

Hospitality setup: Set general options for POS (see Hospitality Setup).

Hospitality types: Create the restaurant layout and settings, create types within the restaurant, such as different areas: Bar, Main and Patio (see Hospitality Types).

Restaurants: Manage restaurants, settings for terminals, allowed payment types, menu types, and suspended transaction can be found here (see Restaurant Configuration).

Printing stations: Set up printers (see Station Printing).

Station routing: Set hospitality types (POSs in different areas within restaurant) to printers (see Station Routing).


Hospitality POS menus: Create, edit and preview Hospitality button grid menus (see Hospitality POS Menus ).

To create and preview touch button grid layouts go to Store setup > Look and feel > Touch button layouts

Hospitality menu groups: Button grid menus grouped together for ease of use when assigning them to the touch button layout on the POS (see Hospitality Menu Groups).
