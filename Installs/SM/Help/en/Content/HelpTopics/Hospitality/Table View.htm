Table View

The table buttons have different colors and text depending on their serving status. The following table lists all available states with the respective color and text.

Text Color Description
Available Dark Green The table is available. Enter an empty table by pressing a green square twice.
Guests seated Lighter green Guests have been seated to the table
Order not printed Yellow An order has been taken but not sent to the kitchen printer or items have been added since last printing.
Order confirmed Red All orders have been sent to a printing station.
Unavailable Gray The table is not in use
N/A Blue Another staff member owns the current transaction on the table
  • The three tabs at the top of the screen are different hospitality type views; Bar area, Rose room, Patio
  • The blue strip of buttons on the right of the screen are operation buttons such as “Seat Guests” or “Log Off.”
