Main Hospitality Dialog Look and Feel

After setting all the configurations in Hospitality Setup, Hospitality Type, Hospitality POS Menus and Restaurant Configuration when starting the LS One POS, the main Hospitality dialog will be visible after logon instead of the default LS One POS retail dialog.

In the image below, you can see how the POS main table view is set through Hospitality types on the Site Manager. To change the configurations, go to

Site Manager > General setup > Hospitality > Hospitality types

The look of the main Hospitality dialog is controlled from the Hospitality type configuration (see Hospitality Type). The number of buttons window tabs/buttons at the top of the screen is set by how many hospitality types are selected on the current terminal (see Terminal settings). The look of the buttons is set in the hospitality type’s Top POS menu id. The button grid on the right is set in the POS Logon menu id setting and the number and description of the table buttons is set in the Dining table layout id settings.
