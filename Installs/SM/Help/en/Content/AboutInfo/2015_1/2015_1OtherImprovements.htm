Other Improvements and Additions

Settings

A setting has been added to SYSTEMSETTINGS table that disables replication action creation. It is called 'DisableReplicationAction' and it is set to true for databases created with LS One In-a-Box. When this setting is true no replication actions are created in database so this should only be true when the POS is only connecting to the local database. There is no UI in Site manager.

License service

The LS One license service malfunctioned and caused problems in accessing LS One POS at a few locations. The steps we have taken to prevent this from happening again were the following: Some error result codes are ignored and an 'OK' is returned instead. These were the error codes we encountered when the license service became faulty. Concurrency mode on the license service was changed from 'single' to 're-entrant' and Instant context mode from 'single' to 'per session'.

LS One Express
  • Security improvements to LS One Express hosted database creation and login mechanism.
  • Release of LS One Express partner and customer guides that explain the setup and usage of LS One Express in detail. The guides are available on the LS Retail partner portal - login required.
Additions to online help topics
  • "How to's" added on the configuration on various different types of discounts
  • Information about image format of logos printed on receipt added to Form settings tab
  • Information about Dashboard and Dashboard ticket selection dialog has been added
  • Detailed information about the Financial report and the numbers that are in that report added
  • Online help for Company card in Site Manager has been updated
  • Special online help project created for LS One In-a-Box. This help only displays the categories and fields that are relevant to LS One In-a-Box. More procedure help will be added in the future.
