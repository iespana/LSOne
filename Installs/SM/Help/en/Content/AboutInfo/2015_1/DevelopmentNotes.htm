Addition of optional service

Partners can now create a service for address lookup.​

POS legacy translation file removed

POS Legacy translation file has been phased out. All texts that were still being used from that file have been moved to the appropriate translation files. All translated languages for each text that were available to LS One team at the time of the phase out were also moved.
