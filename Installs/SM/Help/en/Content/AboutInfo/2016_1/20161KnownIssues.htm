2016.1 Known Issues

Inventory

• If an Inventory template is created using 'All stores', worksheets are created for all store for the first template but not for others. Creating templates for one store at a time is OK.

• Two lines for the same variant item can be added to a transfer request and order. This should not be allowed and causes duplication of lines when transfer order is created from request.

Reports

An error was found in the financial report when payment types do not have same IDs as in demo data. This is only found in the Financial report not the Financial report by statement or user. New report files have been created and can be downloaded and imported through Site Manager.

Replication data

Subjobs for 4 new tables are missing from the replication data package. Subjobs should be added to Item, Preload POS and Preload store jobs.

  • SPECIALGROUPS
  • SPECIALGROUPITEMS
  • DIMENSIONATTRIBUTE
  • DIMENSIONTEMPLATE
