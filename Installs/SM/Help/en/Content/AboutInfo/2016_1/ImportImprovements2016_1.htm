Import with Excel and CSV files

Item, replenishment and stock counting import has been added back into Site Manager and fixed to handle the item variant framework changes.

Stock counting lines can now be imported from CSV files as well as excel files. This was added to support import from hand held stock counting devices.

CSV import profile

The CSV import profile maps columns from CSV file to variables recognized by LS One. At least one CSV profile must exist and be set as default for CSV import to work.

Stock counting line import (CSV and Excel)

Stock counting lines can now be imported from both excel and CSV files. Several files can now be imported at the same time to a single journal.

Item export/import

Item export now exports all retail and variant items displayed in the item list view. If variant items are not shown and a header item is displayed in the item list then all variant items for that variant header are exported. Variant headers are never exported.

New variant items can now be created using the item import. ItemID and headerID for the new item need to be added to the import file. Variant headerID for new variant items does not need to exist in the database, if a new headerID is added the variant header is created using the given headerID and the item description. Variant items imported in this way are not created using dimensions and attributes and the variant header will not display any dimension information.

Replenishment information export/import

Replenishment information can now be exported to an excel file, modified and imported back into the system.

In this release when exporting to replenishment template all retail and variant items displayed in the item list view are exported, not only those already having replenishment settings.

Import result view

The import result view has been updated. Now all errors are displayed in a special list view below the regular result view with line number from the excel file, the ID of the item that was not imported, item description and an error text. This list can then be exported to excel.
