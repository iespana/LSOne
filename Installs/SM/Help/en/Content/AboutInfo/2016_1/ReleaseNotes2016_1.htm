Release Notes 2016.1

The 2016.1 version of LS One includes improvements to users view and inventory management. Creation of purchase orders, stock counting and adjustments has been improved and display of information has been revised. Translations of LS One have also been improved and we now provide LS One in several languages.

The big changes

Following is a list of all changes we have made since last release:

  • Inventory improvements in 2016.1
  • Import with Excel and CSV files
  • User interface improvements 2016.1

 

Smaller changes

Improvements to Users view

Users view has been updated and now shows more information than before about each user, search bar has been added, icon has been removed and if a user is disabled the row is highlighted.

Dimension templates

The user is now able to save dimension templates from dimensions on variant headers.

These dimension templates have a set of attributes and can be reused when creating new variant headers and items.

A 'Save dimension as template' feature has been added to the variant header card. This means that you can save dimensions that have been selected on a certain variant header as templates and reuse them for other items. You can also during this process create new dimensions and attributes that can be saved as templates as well.

There is however no view where all dimension templates can view and edited, except for when you select them as dimensions on a variant header.

Infocode management

Infocode views have added back to the Site Manager and have been update to reflect the new variant framework.

Infocodes that are connected to a retail group are now triggered whenever an item from that group is sold on POS.

When an infocode that is selling an item is configured as "No charge" the price is set to zero. Previously the POS was retrieving the base price for the item.

Translations

Strange Icelandic translations in Unit dialog have been fixed.

The LS One translation processes for partners has also been updated and a new guide and tool published in the partner development help
