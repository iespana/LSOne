User interface improvements 2016.1

We are excited to present the new Site Manager. We introduced a new color scheme for the product, and new images and icons for the different functionalities. To enhance usability, the ribbon was reorganized to create a better user experience. See more detailed information about the new ribbon tabs here below.

The dashboard look and feel was also redesigned to fit into the new color scheme.

Ribbon - Site Manager top navigation

The ribbon, the Site Manager's top navigation was also rearranged for usability improvement. Now each tab on the ribbon contains all functionality that relates to its category.

UX improvements

  • Alignment of text fields fixed in Allowed payment types > Settings tab.
  • Tab order in fixed in Authenticated SQL servers dialog at Site Manager log in.
  • Now Variant description is displayed in Inventory look up dialog header in POS.
  • Various dialogs have been changed to be centered to the Site Manager when they are opened. They were centered to the screen before.
  • Double click to edit attributes and dimensions now works from New retail item dialog > 'Dimensions' tab as well.
  • Double click to edit now works in dimension and attribute list views in dimension dialog on Variant header tab on retail item card. Only new dimensions and attributes are editable.
  • General tab of the Infocode card has been updated to create more space for the fields on the tab
  • New retail item dialog made to fit resolutions with height of 768 pixels
  • POS Menu header dialog made to fit resolutions with height of 768 pixels
  • Tab order in New subcode dialog has been fixed
Last updated: 6th October 2017
