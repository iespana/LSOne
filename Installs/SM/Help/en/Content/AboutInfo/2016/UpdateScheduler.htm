Update Scheduler data to 2016

When you receive a new version LS One you can either

  • delete the old scheduler data and import the new replication data package
  • or you can upgrade the old one.

To be able to upgrade you will need the following information

New tables

Create both normal and action jobs for these new tables

  • RETAILDEPARTMENT
  • RETAILDEVISION
  • RETAILGROUP
  • RETAILITEM
  • RETAILITEMDIMENSION
  • RETAILITEMDIMENSIONATTRIBUTE
  • RETAILITEMIMAGE
  • PERIODICDICOUNT
  • PERIODICDICOUNTLINE
  • SPECIALGROUPS
  • SPECIALGROUPITEMS
  • DIMENSIONATTRIBUTE
  • DIMENSIONTEMPLATE

Scheduler data update

To update your scheduler data from LS One version 2015.1 to version 2016 you will first have to do a Read design. Then you will have to remove old subjobs from the jobs and at last create subjobs for the new tables and add them to the jobs being sent.

Run Site Manager version 2016 and log in to your database.

When you do this the first time your database will be upgraded to 2016 version. New tables and fields are added to your database.

1. Do a Read design
  1. Go to General setup > Replication > Distribution location. Open the location from where you manage your replication and scheduling.
  2. In the right hand navigation press Read design

  3. Select Tables and fields option and press OK.
  4. Now the Data Director will run through your database design and update it to include all new fields and tables.
2. Add new subjobs

Add both action (A-) and normal (N-) jobs for all the new data tables listed above. On further information on how to do so read Subjobs

3. Delete unused subjobs from jobs

Delete all the subjobs for the old data tables from the jobs and delete the subjobs also.

4. Add new subjobs to the jobs

Add the new action and normal jobs to the existing jobs. Read this topic: Jobs for more information on how to create and manage jobs.
