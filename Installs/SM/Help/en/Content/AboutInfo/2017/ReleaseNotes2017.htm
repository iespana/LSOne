Release notes 2017

In this version we have released the much anticipated Mobile POS for LS One. Included we have also added new service items in addition to standard retail items, major user interface improvements in the Site Manager and POS as well as started our work to support Power BI (LS Insight) as our future platform for Business Intelligence. This and much more will pleasantly surprise all our LS One partners when experiencing the latest release.

OMNI Mobile POS

Releasing the first omni product in our broad ecosystem strategy, the Mobile POS is a major milestone for LS One. LS One platform is now a fully stateless platform that can support any number of devices whether standard POS or Mobile POS to connect at the same time. The Mobile POS runs on both Android and IOs. We look forward to get our partners and customer feedback on this very exciting development and more omni products will follow soon.

 

Inventory enhancements

Our partners have spoken and as always we have listened. Service items that allow customer to sell services by e.g. hour or work is now part of LS One. It complements the already support for retail items and variant headers that forms the basis anything being sold.

Read further about inventory enhancements

 

Power BI (LS Insight)

More and more customers are moving toward 3rd party BI tools as an alternative to standard reports. In LS Retail, our LS Insight product runs on Microsoft Power BI and LS One has beta release to selected partner. In this release we support sales and forecast reports together with a user friendly data schema to create user specific data visualizations. The official release for LS One Insight will be in release 2017.1

 

User experience

Our commitment to improve our LS One user experience is continuing. We have moved things around and so now all menus are available in the top ribbon in a much more intuitive layout. We have updated the styles in the POS to be much more aligned with the work that we have done in release 16.1 in the Site Manager.

For more information about user experience changes read this

 

Site Manager license changes

In this release we are introducing changes to the license mechanism of the Site Manager.

Instead of getting a separate license key for the Site Manager and LS One POS now both systems are licenses through the same account key.

This change will make the installation and activation of the LS One POS system much more efficient as well as the license maintenance and information for our customers and partners.

Please see information on how to activate the license in the Site Manager with this new release and how to activate a demo license as well here.

 

Other improvements

Many more improvements have been done in this release such as functionality to email receipts from the POS and other receipt printing enhancements and more.

We encourage you to read through our release notes to see what changes and improvements we have made in this fantastic release and we trust you will like it as much as we do.
