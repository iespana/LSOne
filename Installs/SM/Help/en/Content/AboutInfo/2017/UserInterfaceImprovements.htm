User interface improvements

In this release we have continued our work in improving the user interface of the LS One POS system. In this release we focused primarily on the LS One POS but also some changes were done in the Site Manager as well.

The most visible change for our users is the removal of the left hand navigation from the Site Manager. All operations that were available there have been moved to the ribbon.

We would also like to point to the spotlight search feature that has been available in the Site Manager from the beginning. This search provides an access to operations as well as items, customers, users and more.

LS One POS user interface improvements

The look and feel of the POS login from, dialogs and buttons has been unified with the changes done in Site Manager last release.

The hospitality look and feel in regards to the table layout and split bill and table transfer dialogs has been completely updated within our demo data and Hospitality data pack that can be imported.

Default system styles for the numpad in the POS and totals box have been reviewed and updated. To get the new system styles an option to "Reset system styles" has been added to the Styles view

The touch button layouts provided in demo data were reorganized and the new color palette applied to it. These layouts can be imported to other databases by importing the Touch button layouts default data package.

Site Manager user interface improvements

View report button (under Tools menu) now displays all reports that have been imported into the system. When new reports are imported the Site Manager needs to be restarted for them to be added to the list.

All tool tip texts in the ribbon in the Site Manager have been reviewed and updated.

Right-click menus in many views within the Site Manager have been updated to accurately represent all actions available.

Edit discount button was added below the periodic discount priorities list view. Move up and move down was added to the right click operations in the periodic discounts priorities list view.

Financial report by statement is now displayed on Show report button in Posted statements

"POS terminals" tab on the retail item view has been renamed to "POS settings"
