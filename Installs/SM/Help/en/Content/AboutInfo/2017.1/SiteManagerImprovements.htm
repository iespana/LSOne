Site Manager improvements

In this section general improvements that mostly or only affect the Site Manager are explained. Some changes may affect both the Site Manager and the LS One POS. For smaller improvements and bug fixes please see the other improvements section of the release notes.

Image bank

A central place to store all images (except for item images) has been created and is called Image bank. The image bank can be accessed in the ribbon under the tab Site and Look and feel. The Look and feel section of the ribbon was also modified to accommodate this new option.

When the LS One system is upgraded a migration happens that moves the images that already exist in various functionalities into the image bank. The following is a list of the images that are moved:

  • Logos for all touch button layouts
  • Receipt logos from all stores
  • Images from all buttons on POS layouts

When the images are moved a reference to the new image bank is placed in each of these places instead of the image itself. This means that the same button or logo can be used in many places and when the image needs to be updated it only needs to be updated in the image bank in one place and the change will take effect where ever the image is being used.

The Image bank view has a search bar which can be used to search for a description or type of image. The dialog where the images are added to the touch button layout, buttons and receipts now also has a search bar and a new image can be added directly through the dialog or added beforehand into the view.

If an image is being used it cannot be removed but it can be edited.

Default form profile changes

The default form profile functionality within Form profiles has been improved in this release. The default form profile can now be edited to some extent.

A new form type (created by partner for example) can be added to the default form profile through the user interface. The layouts can also be changed i.e. a new Voided transaction layout is been created which should be the default receipt for all stores. The Voided transaction form type can be selected, edited and the new layout can be set as the default layout.

A form type cannot be removed from the default form profile nor can a form type be empty i.e. without a layout attached.

In previous releases LS One included a data pack that included all receipts, this has now been replaced by system layouts which can be reset and updated after each release.

Note!!! When resetting the default profile in an existing customer database the previously existing layouts are not removed. The resetting of the default profile will insert all the layouts again and mark them as "system layouts"

Tips!!! Before resetting the default profile remove all form layouts that have not been changed by the customer.

Tips!!! To keep a copy of all layouts prior to resetting the default profile, all the layouts can be selected in Form layouts and exported in one click.

User interface changes

Some general user interface changes have been done in this release. In all instances listed in this section the functionality of the view or card has not been changed but the user interface has been updated and made more user friendly.

Vendor view has been changed to create more space for information and a search bar has been added to the items list for the vendor

Suspended transactions view has been updated. A search has been added to the view to make it easier for the user to find suspended transactions.

Store card has been changed to add more space for additional tabs on the card. The address field was moved to the Settings tab but other tabs were not changed.

Gift cards view has been updated. A search bar has been added to the view where various search parameters can be used to filter the list of gift cards.

Credit vouchers view has been updated and now contains a search bar to search for credit memos. The list view has also been updated

Column names on the income / expense tab on the store card have been set to the correct casing

Forms layout view has been changed to have a search bar to make it easier to find form layouts
