2014 Version Numbers

This release is a major release of LS One. The version numbers are as follows:

  • LS One POS - Version 9.4.2014
  • LS One Site Manager - Version 9.4.2014
  • LS One Site Service - Version 9.4.2014
  • LS Printing Station - Version 9.4.2014

In this release, version numbers of all the above applications have been synchronized to a common level.

See Release notes for a list of improvements included in this release.
