Site Manager Licenses

Activation

When the Site Manager is activated you have a choice between activating the application over the internet, which is the most common procedure, or by phone. The phone option is available mainly in case there is no internet access, and in case the internet activation is not successful for some reason. When you run the application a window with these two options appears. 3.1.1 Image: Site Manager Activation

Activating the Application over the Internet

If Activation over the internet, is selected, the user is asked to enter a customer key. This key is supplied by the vendor of the Site Manager.

In case the key has not been provided you need to contact the vendor to obtain the key, as stated in the explanation in the Site Manager activation dialog.

Depending on the kind of license the user has, one or more licenses appear on the list but only one should be selected. The customer may, for instance, have a Base Pack license and also a Hospitality license on top of the Base Pack.

Select the Finish button to conclude the activation of the Site Manager.

Activation by Phone

Select the Activation by phone option if you do not have internet access or in case problems arise while you are activating the application. This gives you access to a vendor or salesperson that can provide the necessary information to activate the application. Enter the customer code you have already got or get this from the vendor over the phone. When the code has been entered another number appears in the next field.

The Code value needs to be provided to the Site Manager provider. After you have entered the second number, the following codes can be entered and the salesperson provides them to the user:

  • Instance code
  • License code
  • Security code

If these codes are correctly filled select the Finish button and conclude the activation.

Conclude the process and log in to the Site Manager.

Deactivation

If there is need to update the hardware the Site Manageris running on you first need to deactivate the Site Manager on the current machine to free the license so it can be activated on the new machine.

 

Deactivation

To do this you launch the Site Managerapplication and in the log in screen open Security > Site Manager license... > Deactivate Site Manager...

The Site Managerasks if you are sure you want to deactivate: Press yes.

When you have done this the Site Managerhas been deactivated and the application will shut down.

The next time the Site Manageris run it will call for Activation.
